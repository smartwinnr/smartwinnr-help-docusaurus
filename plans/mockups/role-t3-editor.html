Sample article

1. Open the Quiz module

From the LMS sidebar, open Quiz → click + New Quiz.

2. Add a title and description

Use Title Case. Keep the description under 160 characters.

3. Add questions

Choose MCQ, True/False, or import from a CSV. See Question Types & Features → CSV import.

Privilege-gated rows show their privilege key as a small pill (e.g. quiz). With PRIVILEGE_GATING_ENABLED = true, an editor whose org doesn't have the quiz privilege would not see the Quiz module at all in the sidebar.
