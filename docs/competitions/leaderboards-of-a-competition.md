Competition can have multiple leaderboards. You can view these leaderboards at an individual or team level.

Individual Leaderboard – When you create a competition, by default a leaderboard called Individual Leaderboard get created as part of the competition. This leaderboard will have all the players from the competition added to it. Thus it shows the relative rankings of all individuals from all teams who are participating in the competition. It takes into account the actual score of every individual.

![](/img/competitions/helpscout-image-491cfc68.png)

Team-wise Leaderboard – If the players in the competition are part of different groups, you can create a Team leaderboard for the competition. The team-wise leaderboard shows the rank and performance of each team that is participating in the competition.

![](/img/competitions/helpscout-image-45141c2c.png)Custom Leaderboard: With a custom leaderboard, you can select a few participants and display their scores and performance in a separate leaderboard. This allows you to view and compare the performance of the selected participants separately.  

![](/img/competitions/helpscout-image-873b3b91.png)

It is mandatory to create at least one team leaderboard while creating the competition and add groups to it.  

Automated Leaderboard: Using the leaderboard, you can basically set a time interval with which the leaderboard will be created automatically.

![](/img/competitions/helpscout-image-1634e5c7.jpg)

In the Creation Mode, select Automated and choose the frequency with which you want to create a leaderboard. The frequency can be daily, weekly, and monthly which will be created within the time duration that you set for.

When the Creation mode is selected as Automated and the frequency is selected with the time duration, select the users you want to add.

![](/img/competitions/helpscout-image-141c8952.jpg)

Select the users you want to add and click on Save icon to create a new automated leaderboard.
