## How to Add a Recording to a SmartFeed

SmartWinnr allows users with the **EDITOR** role to add **recordings** to SmartFeeds. This enables you to share audio, video, or screen recordings directly with your audience, making communication more engaging and informative.

Step-by-Step Instructions

#### 1\. Log in to SmartWinnr

* Sign in to the **SmartWinnr Admin Portal** with your credentials.

#### 2\. Access SmartFeeds

* Navigate to **EDITOR > LEARNING AND KNOWLEDGE > SmartFeeds > View All SmartFeeds**.
* The SmartFeeds listing page will open.

#### 3\. Create a New SmartFeed

* Click **Add New SmartFeed**.
* Select the appropriate **Division** for the SmartFeed.
* Choose **Recording** as the content type.

#### 4\. Enter SmartFeed Details

* Add a **Title** for the SmartFeed.
* Choose the type of recording you want to include

![](/img/helpscout/editors/how-to-add-a-recording-to-a-smartfeed-1.jpg)

Then select the type of recording you would like to create. SmartWinnr supports the following recording options:

* **Screen Recording**

    Allows you to record your screen activity. This is useful for demonstrating workflows or explaining step-by-step processes such as data entry or system navigation.

* **Video Recording**

    Enables you to record a video using your camera. This option is ideal for personally explaining processes, sharing announcements, or delivering guided instructions.

* **Audio Recording**

    Allows you to record voice-only audio. This is suitable for short explanations, instructions, or quick updates.

After completing recording , click **Save SmartFeed** to successfully create and publish the recording-based SmartFeed.
