To build a structured learning experience within a SmartPath, you first need to create the modules that will be included in it. A module typically contains different segments such as SmartFeeds, Quizzes, Surveys, Coachings, Action Item, etc.

Follow the below steps to create modules for your SmartPath:

## **Step 1: Create Module**

* Go to **EDITOR > LEARNING AND KNOWLEDGE > SmartPaths > View All Modules**.

    This will display the list of all modules that have been created within your business unit.

![](/img/helpscout/editors/how-to-create-a-module-for-smartpath-1.png)

* Click on the **Create Module** button located at the top right of the page. It will take you to the following page.

![](/img/helpscout/editors/how-to-create-a-module-for-smartpath-2.png)

* Enter the following details:

* **Business Unit:** Choose the business unit for which the module is being created.
* **Module Title:** Give a clear and descriptive title for the module.
* Module Code: You can give a module code which helps you to uniquely identify the module.
* **Description:** Provide a short summary of what the module covers.

2. Optionally, you can customize the module’s appearance by updating the **Cover Image** and **Color** of the module card.
3. After entering the details, click **Save Module**. You will then be taken to the next step to add segments to the module.

## Step 2: Adding Segments to the Module

The next step is to add segments like _Quizzes, SmartFeeds, Coachings,_ _Surveys_ to the module.

To create a segment in the module, click **Add Segment** at the top right.

![](/img/helpscout/editors/how-to-create-a-module-for-smartpath-3.png)

It will take you to the following page, where you can select the type of segment and create the new segment for your module.

To create a new segment,

* choose the segment type
* provide a name to the segment
* fill up the points on completion of this segment

then click **Create**.

![](/img/helpscout/editors/how-to-create-a-module-for-smartpath-4.png)

_It will take you to the page where you have to enter all the details related to creating a particular segment created._

Check out the links below to know how to create each of the segments:

### **Duplicating the existing content as Segment:**

Alternatively, you can duplicate an existing segment. In order to duplicate existing segments, you just have to simply _click_ on the **'Duplicate'** button that is to the right-hand side of a required segment.

[![](/img/helpscout/editors/how-to-create-a-module-for-smartpath-5.png)](https://s3.amazonaws.com/helpscout.net/docs/assets/5fcc5d22d580ce55a38b4a61/images/601532e1a4cefb30ae5c5910/file-Jooh7k8ls4.png)

It will open the following pop-up asking you to confirm if you would like to duplicate the segment:

[![](/img/helpscout/editors/how-to-create-a-module-for-smartpath-6.png)](https://s3.amazonaws.com/helpscout.net/docs/assets/5fcc5d22d580ce55a38b4a61/images/60154f421f25b9041beb99c9/file-fj60FeElWP.png)

Click on **'YES'** and the _segment gets added_ to the module successfully.

It will get displayed in a module as shown in the screenshot below:

![](/img/helpscout/editors/how-to-create-a-module-for-smartpath-7.png)

You can add **multiple segments** to a module. Once the segments are added, you can update the sequence in which you would like to arrange these segments in the module. C_lick_ on the **'Reorder Segments'** button at the top right corner to re-order the segments in modules.

![](/img/helpscout/editors/how-to-create-a-module-for-smartpath-8.png)

**_You can drag and drop the segments in order to re-order them._** **Once you are done re-ordering,** **_click_** **on the 'Done' button.**

**Manage Rule:**  You can also apply **access rules** for segments, such as requiring completion of a previous segment or achieving a specific score before moving forward.

For example, you could choose to give access to the next segment in the module depending on:

* _If the previous segment is completed or not_
* _On a particular date_
* _And, if the participant has scored a certain threshold score from the previous segments._

**_Please note that the completion rule or score rule can be applied only from the second segment of a segment_**

To apply rules, _click_ on the '**Manage Rules'** button that is beside each segment.

![](/img/helpscout/editors/how-to-create-a-module-for-smartpath-9.png)

It opens the following pop-up:

![](/img/helpscout/editors/how-to-create-a-module-for-smartpath-10.png)

**1. By** **Day****:** If you select the 'Type' as By _Day_, then you have to select a day and time at which the segment can be accessed. Day starts from 0 and other day is the first day of each batch for the SmartPath. Once you enter these details, _click_ on the 'Add Rule' button to add the segment successfully.

**[![](/img/helpscout/editors/how-to-create-a-module-for-smartpath-11.png)](https://s3.amazonaws.com/helpscout.net/docs/assets/5fcc5d22d580ce55a38b4a61/images/601542a71f25b9041beb99b9/file-kHSoB0CJlr.png)**

**2\.** **On Completion:** If you select 'On Completion', _you have to select a segment_. _Only upon completion of the selected segment, the participants will be able to access the next segment._ _Click_ on 'Add Rule' once done.

**[![](/img/helpscout/editors/how-to-create-a-module-for-smartpath-12.png)](https://s3.amazonaws.com/helpscout.net/docs/assets/5fcc5d22d580ce55a38b4a61/images/6015391cac2f834ec5383783/file-PmOern9Ttb.png)**

**3\.** **By Score****:** If you select this option, you have to choose a segment in _Segments_ and set criteria. You can _set a Score_ and _a Comparator_ \- _'Less Than', 'Greater Than', 'Equal To'._

Only upon achieving the threshold score that is set for this segment, the participant will be able to access the next segment.

For example, you can say that only if someone gets a score greater than or equal to 85, in the previous segment, he will be able to access the next segment.

**[![](/img/helpscout/editors/how-to-create-a-module-for-smartpath-13.png)](https://s3.amazonaws.com/helpscout.net/docs/assets/5fcc5d22d580ce55a38b4a61/images/6015439f6867724dfc6edbfa/file-v59Ttya7Za.png)**

_Click_ on Done once you have finished filling the filter rules.

Congratulations!, now you have successfully created a module for SmartPath.
