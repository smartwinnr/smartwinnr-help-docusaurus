### Competition Leaderboards

A competition can include multiple leaderboards, which can be viewed at either the **individual** or **team** level.

* **Individual Leaderboard:**

    By default, when you create a competition, an **Individual Leaderboard** is automatically generated. This leaderboard includes all participants in the competition and displays the **relative rankings of individuals** across all teams. Rankings are based on the **actual scores** of each participant.

![](/img/helpscout/editors/leaderboards-of-a-competition-2.png)

**Team-wise Leaderboard:**

If participants in the competition belong to different groups, you can create a **Team Leaderboard**. This leaderboard displays the **rankings and performance of each team** participating in the competition, providing a clear view of team-level progress.

![](/img/helpscout/editors/leaderboards-of-a-competition-3.png)

* **Custom Leaderboard:**

    A **Custom Leaderboard** allows you to select specific participants and display their scores and performance in a separate leaderboard. This feature enables you to **compare the performance of the selected participants independently** from the main competition leaderboards.

![](/img/helpscout/editors/leaderboards-of-a-competition-4.png)

It is mandatory to create at least one team leaderboard while creating the competition and add groups to it.

How to create the leaderboards:

Below given is the settings available for creating the leaderboards.

![](/img/helpscout/editors/leaderboards-of-a-competition-5.png)

**Leaderboard Name:** Enter a name for the leaderboard.

**Leaderboard Description:** Provide a brief description of the leaderboard.

**Type:** Select the type of leaderboard, such as **Individual**, **Team**, **Scorecard**, etc.

**Creation Mode:**

* In the Creation Mode, select Automated and choose the frequency with which you want to create a leaderboard. The frequency can be daily, weekly, or monthly, which will be created within the time duration that you set.
* When the creation mode is selected as 'Automated' and the frequency is selected with the time duration, select the users you want to add.

**Show Score Breakdown For:** This is to restrict the visibility of the leaderboard to the manager, whether they should see all the members, or all their reportees, or the immediate reportees' scores only.

**Visible To:** This setting allows you to control **who can view the leaderboard**. You can choose to make it visible to:

* **All participants** in the competition
* **Only the members** added to the leaderboard
* **Members and their reportees** who are part of the leaderboard

**Advanced Quiz Setting:**

* **Show Time:** Display the quiz completion time on the leaderboard.
* **Show Ranking:** Show ranks for participants or teams.
* **Show Score:** Display scores achieved by participants.
* **Show Your Own Score Only:** Participants will see only their own scores; other members’ scores will remain hidden.
* **Show Scores with Decimal Points:** Display scores including decimal points.
* **Show Ranks Up To:** Define the number of ranks to display. For example, setting it to **10** will show only the top 10 participants.
* **Meta Tag Type:** Optionally display a specific **meta tag** value of participants on the leaderboard.
* **Scoring Duration:** Choose whether scoring should be calculated for the **entire competition period** or within a **specific start and end date/time**.
* **Tasks:** Include only scores from specific tasks in the leaderboard. If no task is specified, scores from all tasks will be considered.
* **Tasks to Exclude:** Exclude scores from specific tasks by selecting them from this list

**Select the participants:**

![](/img/helpscout/editors/leaderboards-of-a-competition-6.jpg)

Select the users you want to add and click on **Save** button to create a new automated leaderboard. Click on the Advanced button and apply filters to select specific set of users.

**Options available in leaderboard setup**

![](/img/helpscout/editors/leaderboards-of-a-competition-7.png)

* **Change Sequence**: Select this to change the sequence of the leaderboards. It will open the below popup where you can drag and drop the available leaderboards to correct order.

![](/img/helpscout/editors/leaderboards-of-a-competition-8.png)

* **Manage Groups**: It helps you to set the group icon images and cover image of the groups so that they will be reflected in the leaderboards in the user view. Click on **Change Icons** tab and click on the **Choose File** to select the file from your system or click on the **Choose from suggested Icons** to select the icons available in the SmartWinnr system.

![](/img/helpscout/editors/leaderboards-of-a-competition-9.png)

Click on the **Change Overview Image** to change the cover image of the leaderboards. Click on the **Choose File** to select the file from your system. Click on the Show Image to see the image selected.

![](/img/helpscout/editors/leaderboards-of-a-competition-10.png)

* **Duplicate:** Use this option to create copies of an existing leaderboard. You can duplicate a leaderboard multiple times, and each new leaderboard will be created with the same configuration as the original.

When you click **Duplicate**, a pop-up window opens where you can:

1. Select the competition name from the **Select Competition** drop-down.
2. Choose the leaderboard to be duplicated from the **Select Leaderboard** drop-down.
3. Enable **Duplicate User List** to include the same set of users as the original leaderboard.
4. Specify the **Number of Duplications** you want to create.

![](/img/helpscout/editors/leaderboards-of-a-competition-11.png)

Click on the **Duplicate** button to create the leaderboard.

* **Manage Leaderboard Observers**: Click on this option to select the observers for all the leaderboards in the contest. It will open the below page where you can select the leaderboards and the observers to add.

![](/img/helpscout/editors/leaderboards-of-a-competition-12.png)

Click the **Save** button to save the selected observers. Once saved, the selected observers will be displayed on the **Observers** button corresponding to that leaderboard.

If you want to assign different observers to the leaderboards, click the same button for the respective leaderboard and follow the same steps to add the observers.

![](/img/helpscout/editors/leaderboards-of-a-competition-13.png)

Click on the **Edit** button to edit the current setup of the leaderboard, and click on **the Delete** button to delete the leaderboard.
