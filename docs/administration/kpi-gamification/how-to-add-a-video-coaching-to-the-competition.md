### Steps to Add Video Coaching to a Competition

1. In the **fourth step** of creating a competition, select **Video Coaching** from the **“Select Task”** dropdown.
2. Enter a **name** for the video coaching task.
3. Assign **points** for completing the task.
4. Set the **start and end dates** for the task.

Below is a screenshot illustrating how to add a **Video Coaching** task to a competition:

![](/img/helpscout/editors/how-to-add-a-video-coaching-to-the-competition-1.png)

5. **Choose Participants Manually:** Check the **“Choose Participants Manually”** checkbox to manually select participants for the Video Coaching task. This allows you to **exclude specific users** from the participant list if needed and assign the task to only specific user set.
6. **Create the Task:** Click the **“Create”** button. You will be redirected to a page where you can **create the Video Coaching task** and add it to the competition.

For detailed instructions on creating a Video Coaching task, refer to the relevant articles:

[How can I create a Video Coaching assignment?](https://smartwinnr.helpscoutdocs.com/article/85-how-can-i-create-a-video-coaching-assignment)
