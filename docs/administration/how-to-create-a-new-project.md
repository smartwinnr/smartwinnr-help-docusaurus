### 1.Project Name & Description

![](/img/admin/helpscout-image-acaeec5e.png)Project Name: This is a mandatory field, where the Editor will be specifying the name of the project.

Project description: The editor needs to specify the description of the project.

### 2.Teams

![](https://support.smartwinnr.com/api/v1/attachments/1050)

The editor will be able toﾂ_Select_ﾂTeams from the available teams from the dropdown, for the project to be created.ﾂﾂ

### 3.Create Teams

Clicking on the Create Teams page will open the following page, here the editor would be able to create a new Team group by specifying the values in the following fields:

* Group Name:ﾂThe editor needs to specify the group name, which is to be created.
* Group Description:ﾂThe editor needs to specify the group description here.
* Group Type:

1. ﾂ ﾂ ﾂ Private: This will be limited to specific members
2. ﾂ ﾂ ﾂ Public: This will be available to all the members The editor will be able toﾂ_Select_ﾂTeams from the available teams from the dropdown, for the project to be created.

![](https://support.smartwinnr.com/api/v1/attachments/1051)

After all the above-required fields are filled, click on "Save And Add Members".ﾂﾂ

### 4.Due Date & Time Zone

## ![](https://support.smartwinnr.com/api/v1/attachments/1052)

The editor needs to specify the Due Date for the specified project to be created, also the Time Zone can also be changed. After all the above-required fields are filled, click on "Save And Add Members".ﾂﾂ

### 5.Project Status

ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂﾂ ![](https://support.smartwinnr.com/api/v1/attachments/1053)

The status of the project can be selected from this dropdown. The elements of this dropdown are specified by the admin.

### 6.Sections

Here the Editor can specify the various section that is needed to be included in the project. The editor needs to specify the Due Date for the specified project to be created, also the Time Zone can also be changed. After all the above-required fields are filled, click on "Save And Add Members".

![](https://support.smartwinnr.com/api/v1/attachments/1101)

The status of the project can be selected from this drop down. The Editor can also add more than one section by clicking on the "Add Section" button on the top right.

### 7.Notification Settings

The editor needs to specify the type of notification he/she wants to send the various team members whom the project has been assigned.

ﾂNotification Type: There are two types of notifications that the editor can specify Push Messages, emails.

This would help the team members to be updated with various events related to the project.

ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂﾂ ![](https://support.smartwinnr.com/api/v1/attachments/1055)

After the selection of Notification Type, one more tab is activated namely Notification Trigger, in which the Editor needs to specify when the selected notification will be triggered.

ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂ ﾂﾂ ![](https://support.smartwinnr.com/api/v1/attachments/1056) ![](/img/admin/helpscout-image-2e644014.png)ﾂThe editor can also add more than one notifications for the same project by clicking on the "Add More" on the top right of this section.

After all the fields have been successfully filled, Editor needs to click on the "Save New Project" to create the project.
