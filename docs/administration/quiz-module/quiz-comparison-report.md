The **Quiz Comparison Report** provides a comparative view of multiple quizzes created in SmartWinnr. This report helps you analyze learner performance and track the progress made by quiz participants across different quizzes over a period of time.

To access the **Quiz Comparison Report** in SmartWinnr, follow these steps:

Navigate to **Editor > Learning and Knowledge > Questions and Quizzes > Reports > Quiz Comparison Report**.

This will open the Quiz Comparison Report page, where you can view and analyze the data.

![](/img/helpscout/editors/quiz-comparison-report-1.png)

You can view quiz progress for participants from a specific country, group, or based on metadata such as designation, category, and other attributes by using the filters available in the **Select Participants** section.

Additionally, you can track the progress of an individual user by using the **Advanced Search** option and searching by the user’s name or email ID.

![](/img/helpscout/editors/quiz-comparison-report-2.png)

### Comparison By Average Score

This section provides a comparison report of the **average scores** achieved by the selected country, group, or individual user across quizzes within the specified date range. By default, it displays the average quiz scores for the **last 30 days**.

![](/img/helpscout/editors/quiz-comparison-report-3.png)

### Performance Across Time by Quiz Category

This section displays the performance of the selected country, group, or individual user for a specific quiz category over a period of time. You can adjust the time frequency to **Daily, Weekly, Monthly, Quarterly, or Yearly** based on your reporting requirements.

Select the desired quiz category from the **Category** dropdown in this section. This view helps you track and analyze your team’s progress in a specific quiz category over time.

![](/img/helpscout/editors/quiz-comparison-report-4.png)
