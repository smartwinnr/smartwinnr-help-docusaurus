## Overview

Send a push notification to remind or inform users about an assigned quiz. This is useful for driving completion rates on quizzes that are already live.

| Detail | Description |
|--------|-------------|
| **Where to find it** | **Quiz View** > **⋮ Menu** > **Send Notification** |
| **Who can use it** | Admin |
| **Applies to** | All quiz types |

## How to Send a Quiz Notification

1. Navigate to the quiz view page.
2. Click the **⋮** (more options) menu.
3. Select **Send Notification**.
4. Review the notification details. The notification is sent to all users who have the quiz assigned.
5. Confirm to send the notification.

> **Tip**: Use notifications strategically — sending too many reminders can reduce engagement over time.

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Users report not receiving the notification | Check that the users have push notifications enabled on their devices. |
| Send Notification option is grayed out | The quiz may not be in an active state. Activate the quiz first. |
