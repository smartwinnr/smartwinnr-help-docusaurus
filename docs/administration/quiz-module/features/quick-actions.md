## Overview

The quiz view menu includes several lightweight actions for managing and maintaining quizzes. This guide covers actions that take only a few steps to complete.

---

## Delete a Quiz

Permanently remove a quiz and all associated data.

1. Navigate to the quiz view page.
2. Click the **⋮** (more options) menu.
3. Select **Delete**.
4. Confirm the deletion in the dialog.

> **Note**: Deleting a quiz is permanent and cannot be undone. All participation data, scores, and analytics for this quiz are removed.

---

## Archive a Quiz

Move a quiz to the archive to declutter your active quiz list without deleting it.

1. Navigate to the quiz view page.
2. Click the **⋮** (more options) menu.
3. Select **Archive**.
4. The quiz moves to the archived section and is no longer visible to users.

> **Tip**: You can restore an archived quiz at any time from the archived quizzes list.

---

## Upload Non-Existing User Data

Upload participation data for users who are not yet registered in SmartWinnr. The system creates placeholder records for these users.

1. Navigate to the quiz view page.
2. Click the **⋮** (more options) menu.
3. Select **Upload Non-Existing User Data**.
4. Download the template file and fill in user details and scores.
5. Upload the completed file.

---

## Download Audit Log

Download a log of all administrative actions taken on a quiz, including edits, status changes, and assignments.

1. Navigate to the quiz view page.
2. Click the **⋮** (more options) menu.
3. Select **Download Audit Log**.
4. The log file downloads to your device.

---

## Download Auto Quiz Completion Details

Download a report of users whose quizzes were automatically completed by the system (for example, when a quiz deadline passed).

1. Navigate to the quiz view page.
2. Click the **⋮** (more options) menu.
3. Select **Download Auto Quiz Completion Details**.
4. The report file downloads to your device.

---

## Generate New xAPI ID

Generate a new xAPI activity identifier for a quiz. Use this when you need to re-register the quiz with an external LRS (Learning Record Store).

1. Navigate to the quiz view page.
2. Click the **⋮** (more options) menu.
3. Select **Generate New xAPI ID**.
4. A new xAPI Activity ID is created and displayed.

> **Note**: Generating a new xAPI ID means the external LRS treats this as a new activity. Historical records under the old ID are not affected.
