## Overview

Export questions from a quiz to a downloadable file. Use this to back up quiz content, share questions across teams, or reuse them in other quizzes or external systems.

| Detail | Description |
|--------|-------------|
| **Where to find it** | **Quiz View** > **⋮ Menu** > **Export Questions** |
| **Who can use it** | Admin |
| **Applies to** | All quiz types |

## How to Export Questions

1. Navigate to the quiz view page.
2. Click the **⋮** (more options) menu.
3. Select **Export Questions**.
4. Choose the export format if prompted.
5. The file downloads automatically to your device. It includes question text, answer options, correct answers, and tags.

> **Tip**: Exported files can be edited and re-imported into a new quiz using the question import feature.

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Export file is empty | Confirm the quiz has questions assigned before exporting. |
| Export option is not visible | Verify you have Admin permissions for the Quiz module. |
