## Overview

Activate or deactivate an event quiz to control when users can access and take it. This is useful for live events where you want to open the quiz at a specific moment and close it when the session ends.

| Detail | Description |
|--------|-------------|
| **Where to find it** | **Quiz View** > **Toolbar** > **Activate** / **Deactivate** |
| **Who can use it** | Admin |
| **Applies to** | Event quizzes |

## How to Activate a Quiz

1. Navigate to the quiz view page.
2. Click the **Activate** button in the toolbar.
3. The quiz becomes immediately available to all assigned users.

## How to Deactivate a Quiz

1. Navigate to the quiz view page.
2. Click the **Deactivate** button in the toolbar.
3. The quiz is no longer accessible to users. Any in-progress attempts are saved.

> **Note**: Activating and deactivating is designed for event quizzes. Scheduled quizzes use start and end dates instead.

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Activate button is not visible | This feature is available only for event-type quizzes. Check the quiz type in settings. |
| Users can still access a deactivated quiz | Clear the app cache on the user's device, or wait a few moments for the status to sync. |
