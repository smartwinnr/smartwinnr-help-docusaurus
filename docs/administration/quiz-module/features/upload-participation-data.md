## Overview

Upload participation data to record quiz completions in bulk. This is useful when quiz results are collected outside the platform (for example, from an in-person assessment) and need to be reflected in SmartWinnr.

| Detail | Description |
|--------|-------------|
| **Where to find it** | **Quiz View** > **⋮ Menu** > **Upload Participation Data** |
| **Who can use it** | Admin |
| **Applies to** | All quiz types |

## How to Upload Participation Data

1. Navigate to the quiz view page.
2. Click the **⋮** (more options) menu.
3. Select **Upload Participation Data**.
4. Download the template file if this is your first upload.
5. Fill in the template with user identifiers and their scores.
6. Click **Choose File** and select your completed file.
7. Click **Upload** to submit the data.
8. Review the upload summary for any errors or skipped records.

> **Note**: Uploaded data is merged with existing participation records. Duplicate entries for the same user update the existing record.

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Upload fails with a format error | Verify the file matches the provided template and is in the correct format (CSV or XLSX). |
| Some users are skipped | Check that user identifiers in the file match existing user accounts in SmartWinnr. |
