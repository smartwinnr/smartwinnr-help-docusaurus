## Overview

Access quiz analytics directly from the quiz view page to see participation rates, scores, and question-level performance data. Use this to identify knowledge gaps and measure the effectiveness of your quizzes.

| Detail | Description |
|--------|-------------|
| **Where to find it** | **Quiz View** > **⋮ Menu** > **Analytics** |
| **Who can use it** | Admin |
| **Applies to** | All quiz types |

## How to View Quiz Analytics

1. Navigate to the quiz view page.
2. Click the **⋮** (more options) menu.
3. Select **Analytics**.
4. The analytics dashboard opens, showing key metrics such as:
   - **Completion rate** — Percentage of assigned users who completed the quiz.
   - **Average score** — Mean score across all participants.
   - **Question-level breakdown** — Performance data for each question.
   - **Time analysis** — Average time spent on the quiz.

> **Tip**: Use question-level data to identify frequently missed questions and update your training content accordingly.

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Analytics show no data | Confirm that users have completed the quiz. Analytics populate after submissions. |
| Scores seem incorrect | Check if the quiz allows multiple attempts — analytics may reflect the latest or best score depending on settings. |
