## Overview

Quiz TV displays live quiz results and leaderboards on a shared screen. Use this during events, training sessions, or team meetings to showcase real-time participation and scores.

| Detail | Description |
|--------|-------------|
| **Where to find it** | **Quiz View** > **Toolbar** > **Quiz TV** |
| **Who can use it** | Admin |
| **Applies to** | All quiz types |

## How to Use Quiz TV

1. Navigate to the quiz view page.
2. Click the **Quiz TV** button in the toolbar.
3. A new window or full-screen view opens showing live quiz data.
4. Display this on a projector or shared screen for your audience.
5. The view refreshes automatically as users submit their answers.

> **Tip**: Set a custom Quiz TV background image in the quiz settings to match your event branding.

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Quiz TV screen is blank | Confirm the quiz is active and at least one user has submitted answers. |
| Scores are not updating | Check your network connection. Quiz TV requires an active internet connection to refresh. |
