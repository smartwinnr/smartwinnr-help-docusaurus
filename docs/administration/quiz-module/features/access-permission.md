## Overview

Access permissions control which administrators or managers can view and manage a specific quiz. Use this to restrict quiz visibility to certain teams or roles within your organization.

| Detail | Description |
|--------|-------------|
| **Where to find it** | **Quiz View** > **⋮ Menu** > **Access Permission** |
| **Who can use it** | Admin |
| **Applies to** | All quiz types |

## How to Manage Access Permissions

1. Navigate to the quiz view page.
2. Click the **⋮** (more options) menu.
3. Select **Access Permission**.
4. In the permissions dialog, add or remove users and roles who should have access.
5. Set the permission level (view, edit, or full control) for each entry.
6. Click **Save** to apply the changes.

> **Note**: Changing access permissions does not affect end-user quiz assignments — it only controls admin-level visibility.

## Troubleshooting

| Issue | Solution |
|-------|----------|
| A manager cannot see a quiz in their dashboard | Add the manager's role or user account to the quiz's access permissions. |
| Cannot remove a user from permissions | The user may be the quiz creator. At least one admin needs access at all times. |
