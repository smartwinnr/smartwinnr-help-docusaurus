## Overview

Duplicating a quiz creates a new copy with the same questions, settings, and configuration as the original. Use this when you want to reuse a quiz structure for a different audience or time period without rebuilding it from scratch.

| Detail | Description |
|--------|-------------|
| **Where to find it** | **Quiz View** > **⋮ Menu** > **Duplicate** |
| **Who can use it** | Admin |
| **Applies to** | All quiz types |

## How to Duplicate a Quiz

1. Navigate to the quiz view page for the quiz you want to copy.
2. Click the **⋮** (more options) menu in the top-right corner.
3. Select **Duplicate**.
4. A new quiz is created with the same questions and settings. The title is prefixed with "Copy of" to distinguish it from the original.
5. Open the duplicated quiz and update the title, schedule, and audience as needed.
6. Click **Save** to finalize your changes.

> **Tip**: After duplicating, review the notification and assignment settings — these are not carried over automatically.

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Duplicate option is not visible | Confirm you have Admin permissions for the Quiz module. |
| Duplicated quiz has no questions | Verify the original quiz has questions assigned before duplicating. |
