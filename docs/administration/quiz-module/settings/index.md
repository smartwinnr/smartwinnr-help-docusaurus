# Quiz Settings

This section provides detailed reference guides for every configurable setting in the Quiz module. Settings are organized by functional group.

## Settings Groups

| Guide | Description |
|-------|-------------|
| [Timer & Attempt Settings](timer-and-attempts) | Quiz time limits, multiple attempts, retakes, and score thresholds |
| [Question Delivery Settings](question-delivery) | Randomization, questions per quiz, shuffle options, and skipping |
| [Scoring & Display Settings](scoring-and-display) | Score visibility, pass/fail, correct answer display, and confidence levels |
| [Notification & Communication Settings](notifications) | Assignment notifications, emails, completion messages, and digest emails |
| [Advanced & Integration Settings](advanced-integrations) | xAPI, leaderboards, proctoring, self-registration, and certificates |
| [Automatic Quiz Settings](automatic-quiz) | Delivery schedules, self-activation, and auto-mode configuration |
| [Visual & Media Settings](visual-and-media) | Cover images, thumbnails, and Quiz TV backgrounds |
