## Overview

Visual and media settings let you customize the appearance of your quizzes with branded images. These settings affect what users see in the quiz list, quiz view, and Quiz TV display.

## Settings Reference

### Quiz Cover Image

| Detail | Description |
|--------|-------------|
| **What it does** | Sets the main image displayed at the top of the quiz view page. |
| **Default** | System default image |
| **Applies to** | All quiz types |
| **Prerequisites** | None |

**How to configure:**

1. Open the quiz in **Edit** mode.
2. Scroll to the **Media** section.
3. Click **Upload** under **Quiz Cover Image**.
4. Select an image with a **16:9** aspect ratio.
5. Click **Save**.

> **Tip**: Use a 16:9 image (for example, 1920x1080 pixels) for the best display quality across devices.

---

### Quiz Thumbnail

| Detail | Description |
|--------|-------------|
| **What it does** | Sets the small preview image shown in the quiz list and dashboard cards. |
| **Default** | Cropped from cover image |
| **Applies to** | All quiz types |
| **Prerequisites** | None |

**How to configure:**

1. Open the quiz in **Edit** mode.
2. Scroll to the **Media** section.
3. Click **Upload** under **Quiz Thumbnail**.
4. Select a square image with a **1:1** aspect ratio (25x25 minimum).
5. Click **Save**.

---

### Quiz TV Background

| Detail | Description |
|--------|-------------|
| **What it does** | Sets the background image displayed on the Quiz TV screen. |
| **Default** | System default background |
| **Applies to** | All quiz types |
| **Prerequisites** | None |

**How to configure:**

1. Open the quiz in **Edit** mode.
2. Scroll to the **Media** section.
3. Click **Upload** under **Quiz TV Background**.
4. Select your background image.
5. Click **Save**.

> **Tip**: Use a high-resolution image that works well on large screens. Avoid busy backgrounds that make text hard to read.

## Frequently Asked Questions

**Q: What image formats are supported?**
A: PNG, JPG, and JPEG formats are supported for all image uploads.

**Q: Will changing the cover image affect the thumbnail?**
A: Only if you have not uploaded a separate thumbnail. If no custom thumbnail is set, the system generates one from the cover image.
