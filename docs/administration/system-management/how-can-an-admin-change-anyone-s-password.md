You can change the password of a user by following the below steps.

Go to ADMIN > Manage Users. Click on the “Change Password” button.

Only users with admin role can change the password.

![](/img/helpscout/admins/how-can-an-admin-change-anyone-s-password-1.png)

The password you select needs to confirm with your organizational password policy as defined in the SmartWinnr system.

![](/img/helpscout/admins/how-can-an-admin-change-anyone-s-password-2.png)

The user will not receive any mail on changing the password through this method. You need to communicate the changed password to the user separately.
