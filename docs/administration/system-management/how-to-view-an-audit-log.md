Go to Admin>>Audit Logs, it will open the following page.

You can see the information about the various existing logs. Information about when they were created , various actions, and also the user associated with that event.

![You can see the information about the various existing logs. Information about when they were created , various actions, and also the user a](https://support.smartwinnr.com/api/v1/attachments/1112)

Logs can be viewed for a specific date range by clicking on the date "Date Range".The varied option provided in the date range is shown below

                                                                                  ![Logs can be viewed for a specific date range by clicking on the date "Date Range".The varied option provided in the date range is shown belo](https://support.smartwinnr.com/api/v1/attachments/1113)

Select the required and click on apply, logs between that date range will be displayed.

You can also select the "Action" from the drop-down for filtering the Log. The various actions are as follows

                                                                                   ![You can also select the "Action" from the drop-down for filtering the Log. The various actions are as follows](https://support.smartwinnr.com/api/v1/attachments/1114)

Audit Logs can be filtered using the "Task Name" and also the "User Name".
