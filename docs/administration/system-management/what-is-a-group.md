### What is a Group?

In SmartWinnr, a **Group** is a collection of users that can be created and managed by Trainers/**Editors.**

The Group feature allows editors to assign **quizzes, SmartFeeds, or surveys** to multiple users simultaneously. Groups can be created **as per business requirements**, such as by **zone, region, territory, role, or any custom criteria**, making content assignment more flexible and efficient.

This feature is especially useful when the same content needs to be shared repeatedly with a specific set of users. Instead of selecting individual participants every time, editors can simply select the relevant Group, saving time and ensuring consistency in content delivery.

**Example:**

During a training program, multiple quizzes may need to be shared with the same group of participants to assess their learning progress. Creating a Group allows trainers to easily assign all related quizzes to the same users throughout the training period.

### Benefits of Using Groups

* **Efficient Content Assignment:** Assign content to multiple users in one action.
* **Time-Saving:** Avoid repeated manual user selection.
* **Consistency:** Ensure the same content reaches the intended audience every time.
* **Reusable:** Use the same group for recurring quizzes, surveys, or SmartFeeds.
* **Scalable:** Ideal for training programs, assessments, and large teams.
