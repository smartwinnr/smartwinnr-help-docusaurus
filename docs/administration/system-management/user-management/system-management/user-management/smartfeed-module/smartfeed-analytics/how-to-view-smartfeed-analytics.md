SmartFeed analytics will give you the number of views for each SmartFeed created by you and the ones to which you have access.

Go to SmartFeeds from the left menu bar  Click on ' **SmartFeed Analytics**'.

You will get to see two tabs here.

1. MyÂtab shows the analytics of the SmartFeeds created by you
2. OthersÂtab shows the analytics of the SmartFeeds that are created by the other editors.[![](/img/learning/helpscout-image-fbafa84a.png)](https://s3.amazonaws.com/helpscout.net/docs/assets/5fcc5d22d580ce55a38b4a61/images/5ff0a2d06451e825e3b8f98c/file-lt4UlY0dZA.png)

You can also search for specific SmartFeed from the search box or choose specific dates from the Date box.

By selecting the **image** icon, you can save the SmartFeed Analysis graph as an image for any further reference.

[![](/img/learning/helpscout-image-228cff09.png)](https://s3.amazonaws.com/helpscout.net/docs/assets/5fcc5d22d580ce55a38b4a61/images/5ff0a49d495a886ca7829545/file-c409ei5Uz7.png)

You also have the option to download the report for the SmartFeed view. Click on the ' **Download Report**' button to download the SmartFeed view report.

[![](/img/learning/helpscout-image-c31ad35a.png)](https://s3.amazonaws.com/helpscout.net/docs/assets/5fcc5d22d580ce55a38b4a61/images/5ff0a49d495a886ca7829545/file-c409ei5Uz7.png)In the downloaded report, you can find the list of users who have viewed the selected SmartFeed.

![](/img/learning/helpscout-image-8dda035d.png)

The downloaded report will have the following details in it:

* Name and email id of the user who viewed the SmartFeed
* Title of the SmartFeed which the user has viewed. This is helpful when you download the report of multiple SmartFeeds at once.
* Number of times the user has viewed the Smartfeed
