SmartWinnr allows you to send **Quizzes, Surveys, and Content** to a predefined set of users using **Groups**, making distribution quick and efficient.

Only the Trainer/Editor can send tasks to groups.

While assigning content, you will see a **Group** field where all Groups accessible to the Editor are listed. The Editor can select the required Group based on the need. Once a Group is selected, the list of users associated with that Group will be displayed.Select the required users from the list and click **Save** to successfully assign the Quiz

![](/img/helpscout/editors/how-to-send-quizzes-surveys-and-content-using-group-1.png)

#### Sending Surveys and Content

The **same process** applies when sending **Surveys** or **Content (SmartFeed)**. Simply select the required Group during participant selection and proceed with saving or assigning.

Using Groups helps avoid selecting individual users and ensures faster, more consistent content delivery.
