The concept of audit logs is simple. When a change is applied to a system, it correlates with a change in the system’s behavior. That change should be documented in an audit log.

In SmartWinnr Audit Logs keep a track of the user's activity on the various events that are created, played, etc. For example, a quiz is played by some users on a certain date as shown below.
