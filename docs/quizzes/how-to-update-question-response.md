SmartWinnr allows you to update/correct the question response of  user in a quiz.

On changing the user response, the system automatically re-calculates the scores and updates the revised score for all analytics and reports as well.

You can follow the below given steps to update/correct the user response.

Go to Editor> Questions and Quizzes > Click on the Analytics of the specific quiz.You will be redirected to the analytics page of the quiz.

![](/img/quizzes/helpscout-image-b15502b4.png)

Scroll down to the bottom and click on analytics icon towards the user you wish to update the response.

![](/img/quizzes/helpscout-image-49aef75d.png)

Click on "**Change Response"** against the user response where you want to make the change

![](/img/quizzes/helpscout-image-004d970d.png)

Update the current response  and click on "**Save"**

![](/img/quizzes/helpscout-image-e0b66bb2.png)Then the response will be updated successfully.
