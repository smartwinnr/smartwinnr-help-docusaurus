# Quiz Administration

## Quiz Configurations & Settings

### Managing Default Quiz Settings

To configure default quiz settings:

1. Go to **Admin > Quiz Settings**
2. Modify default configurations including:
   - **Points for correct answers**
   - **Points for wrong answers** (deduction settings)
   - **Time limits and duration**
   - **Attempt limits**
   - **Threshold scores for badges**

### Badge Thresholds

Set minimum average scores required to achieve different types of badges:
- **Bronze Badge**: Set minimum percentage (e.g., 60%)
- **Silver Badge**: Set minimum percentage (e.g., 75%)  
- **Gold Badge**: Set minimum percentage (e.g., 90%)

### Quiz Notification Settings

Configure automatic notifications for:
- Quiz assignments
- Quiz reminders before deadline
- Quiz completion confirmations
- Score announcements

## Opening and Closing Quizzes

### Opening an Inactive Quiz

To activate a quiz that's been created but not yet started:

1. Go to **Editor > LEARNING AND KNOWLEDGE > Competition**
2. Open the desired competition
3. Click on the **"View Tasks"** tab
4. Find the quiz you want to activate
5. Click on the **"Open Quiz"** button

### Closing an Active Quiz

To close a quiz before its scheduled end time:

1. Navigate to the same **"View Tasks"** section
2. Find the active quiz
3. Click on the **"Close Quiz"** button
4. Confirm the closure (note: participants can no longer take the quiz)

### Scheduling Quiz Availability

When creating or editing a quiz, you can set:
- **Start date and time**: When the quiz becomes available
- **End date and time**: When the quiz automatically closes
- **Duration**: How long participants have to complete the quiz once started

## User Management & Permissions

### Giving Access Permissions to Other Editors

To grant other editors access to your quizzes:

1. Go to the quiz you want to share
2. Click on **"Permissions"** or **"Share Access"**
3. Add editor email addresses or usernames
4. Set permission levels:
   - **View Only**: Can see quiz and results
   - **Edit**: Can modify quiz content and settings
   - **Full Control**: Can edit, publish, and manage permissions

### Managing Quiz Participants

#### Sending Quizzes to Specific Groups

1. Navigate to **Quiz Assignment**
2. Select **"Send using Groups"**
3. Choose target groups:
   - Department groups
   - Location-based groups
   - Role-based groups
   - Custom groups
4. Set assignment parameters and send

#### Individual User Management

- **Activate/Deactivate Users**: Control who can access quizzes
- **Reset Quiz Attempts**: Allow users to retake quizzes if needed
- **Manual Score Adjustments**: Modify scores when necessary

## Quiz Duplication & Templates

### Duplicating Existing Quizzes

To create a copy of an existing quiz:

1. Go to your quiz list
2. Find the quiz you want to duplicate
3. Click on the **"Duplicate"** or **"Copy"** icon
4. Modify the duplicate as needed:
   - Change title and description
   - Update questions if necessary
   - Adjust settings and timing
5. Publish the new quiz

### Creating Quiz Templates

For frequently used quiz formats:

1. Create a "master" quiz with standard settings
2. Save as template
3. Use the template for future quiz creation
4. Customize content while keeping standard structure

## Reminder & Notification Management

### Setting Up Automatic Reminders

Configure reminders to improve participation:

1. Go to **Quiz Settings > Notifications**
2. Enable **"Automatic Reminders"**
3. Set reminder schedule:
   - **First reminder**: X days after assignment
   - **Follow-up reminders**: Every X days until completion
   - **Final reminder**: X hours before deadline
4. Customize reminder message content

### Manual Reminder System

For immediate notifications:

1. Select participants who haven't completed the quiz
2. Choose **"Send Reminder"**
3. Customize the message
4. Send individual or bulk reminders

## Question Bank Management

### Organizing Questions

- **Knowledge Categories**: Group questions by subject matter
- **Difficulty Levels**: Categorize as Easy, Medium, Hard
- **Tags**: Add descriptive tags for easy searching
- **Question Types**: Organize by format (MCQ, Fill-in-blank, etc.)

### Import/Export Functions

#### Bulk Question Import
1. Download the question template
2. Fill in questions following the format
3. Include all required fields
4. Upload the completed file
5. Review and approve imported questions

#### Question Export
- Export questions for backup
- Share question banks between teams
- Create question libraries for reuse

### Question Maintenance

- **Regular Review**: Update outdated questions
- **Performance Analysis**: Identify problematic questions
- **Version Control**: Track question modifications
- **Archive System**: Retire outdated questions without deletion

## Security & Access Control

### Quiz Security Settings

- **Time limits**: Prevent extended quiz sessions
- **Attempt restrictions**: Limit number of tries
- **Random question order**: Reduce cheating
- **Browser lockdown**: Restrict navigation during quiz
- **Proctoring options**: Enable monitoring if available

### Data Privacy

- **User data protection**: Ensure compliance with privacy policies
- **Score confidentiality**: Control who can view individual results
- **Anonymous options**: Allow anonymous quiz participation when appropriate

## Troubleshooting Common Issues

### Quiz Not Appearing for Users
- Check quiz status (active/inactive)
- Verify user group assignments
- Confirm quiz publication date
- Review user permissions

### Technical Issues
- **Mobile compatibility**: Test on different devices
- **Browser support**: Ensure compatibility across browsers
- **Media loading**: Verify images, videos, and audio files work properly
- **Time zone issues**: Consider global teams and time differences

### Performance Issues
- **Large quizzes**: Break into smaller sections if needed
- **Media-heavy content**: Optimize file sizes
- **High participation**: Monitor server performance during peak times
