 A user who is having admin access can add the competencies for field coaching.

Below are the steps to create competencies.

Step 1: Select the left menu>>Go to ADMIN>>Field Coaching Settings>>Competencies.

![](/img/coaching/helpscout-image-71705a60.jpg)

Step 2: Add the competencies by creating a group by clicking on the " Add New " icon. 

![](/img/coaching/helpscout-image-dc66f1db.png)

Step 3: By entering the group name and competencies name, you can add the competencies by clicking on the save icon at the end of this page.

By following these steps, the competencies for field coaching will be created.
