An Editor who has access to the Learner Report can see the Report Analytics.

GO TO EDITOR>>REPORTS>>LEARNER REPORT>>BY LEARNER.

![GO TO EDITOR>>REPORTS>>LEARNER REPORT>>BY LEARNER.](/img/helpscout/editors/learner-report-by-learner-individual-report-analytics-1.jpg)

Select the user you want to check the learner report by clicking on the Analytics icon. You can also filter the user by selecting the specific Business Unit, Country, and Group/Team.

![Select the user you want to check the learner report by clicking on the Analytics icon. You can also filter the user by selecting the specif](/img/helpscout/editors/learner-report-by-learner-individual-report-analytics-2.jpg)

You can change the date range to get the specific analytics report for a user. Then you can check the analytics for any particular quiz, survey, coaching by clicking on the Analytics icon beside the activity.

![You can change the date range to get the specific analytics report for a user. Then you can check the analytics for any particular quiz, sur](/img/helpscout/editors/learner-report-by-learner-individual-report-analytics-3.jpg)

Here you will see all the data for an activity of the user like the Quizzes Played, Quizzes Pending, Net Score, and Percentage Score.

![Here you will see all the data for an activity of the user like the Quizzes Played, Quizzes Pending, Net Score, and Percentage Score.](/img/helpscout/editors/learner-report-by-learner-individual-report-analytics-4.jpg)

After scrolling down, you can check the responses under Quiz Response for the quiz questions with the time taken and the correct responses. You can also change the responses as well by clicking on the change response icon.

![After scrolling down, you can check the responses under Quiz Response for the quiz questions with the time taken and the correct responses. ](/img/helpscout/editors/learner-report-by-learner-individual-report-analytics-5.jpg)

In Answer Analytics, you can get the details for total time taken, the average time taken, and correct responses.

![In Answer Analytics, you can get the details for total time taken, the average time taken, and correct responses.](/img/helpscout/editors/learner-report-by-learner-individual-report-analytics-6.jpg)

Scroll down to get the Quiz Wise Performance, where you get the Correct Answers, Incorrect Answers, Net Score, and Percentage Score.

![Scroll down to get the Quiz Wise Performance, where you get the Correct Answers, Incorrect Answers, Net Score, and Percentage Score.](/img/helpscout/editors/learner-report-by-learner-individual-report-analytics-7.jpg)

Similarly, for Survey, you can also check the analytics by clicking on the Analytics icon.

![Similarly, for Survey, you can also check the analytics by clicking on the Analytics icon.](/img/helpscout/editors/learner-report-by-learner-individual-report-analytics-8.jpg)

Here you get the analytics for all the different types of survey questions.

![Here you get the analytics for all the different types of survey questions.](/img/helpscout/editors/learner-report-by-learner-individual-report-analytics-9.jpg)

For Video Coaching, you can check the analytics as well by clicking on the Analytics icon. Here you can check the uploaded video and the competencies on which the user was scored.

![For Video Coaching, you can check the analytics as well by clicking on the Analytics icon. Here you can check the uploaded video and the com](/img/helpscout/editors/learner-report-by-learner-individual-report-analytics-10.jpg)

This is how you can check the user report analytics for any activities participated by the user.
