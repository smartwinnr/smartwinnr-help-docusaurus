> **At a glance** - A Missing Words question allows participants to fill in the blanks by selecting the correct word(s) from a list of options.

A **Missing Words question** enables participants to complete a sentence or phrase by selecting the correct word(s) from a predefined list of options.

## When to use this
Use a Missing Words question when you want participants to demonstrate their understanding of a concept by filling in the blanks in a sentence or phrase. This type of question is useful for:

- Assessing comprehension of key terms or phrases.
- Encouraging recall of specific information.
- Enhancing engagement through interactive content.

## Steps

### 1. Access the Question Bank
Navigate to **EDITOR > LEARNING AND KNOWLEDGE > Questions and Quizzes > Question Bank** in the SmartWinnr editor.

### 2. Initiate a New Question
Click the **Add Question** button located in the top-right corner of the page.

![](/img/helpscout/authored/how-to-create-a-missing-words-question-mr1s1pa8.png)

### 3. Select the Question Type
In the question creation page, choose **Missing Words** from the **Question Type** dropdown.

![In the question creation interface, choose Missing Words from the Question Type dropdown.](/img/helpscout/editors/how-to-create-a-missing-words-question-2.png)

### 4. Enter the Number of Missing Words
Enter the number of missing words in the question.

![Enter the number of missing words in the question.](/img/helpscout/editors/how-to-create-a-missing-words-question-3.png)

### 5. Enter the Question
Type your question or sentence into the **Question** field. Within the text, insert placeholders where each missing word should appear by using the designated placeholder button shown below:
![](/img/helpscout/authored/how-to-create-a-missing-words-question-mr1s8msw.png)

Once the placeholder is placed in the blanks where each missing word should appear, question text will look as follows:

![](/img/helpscout/authored/how-to-create-a-missing-words-question-mr1scomn.png)

### 6. Define Missing Word Groups and Answers
Enter the option texts for each missing word and assign them to the appropriate answer option group. For example, if the question includes only one missing word, there will be a single answer option group (**Group 1**). If the question contains two missing words, create corresponding answer option groups (**Group 1** and **Group 2**).

After entering all option texts for each missing word group, select the checkbox **“Mark this as the correct answer option”** for the appropriate option. This ensures that the system awards the correct score when the user selects the correct answer.

Below is an example for settings with two missing words.

![](/img/helpscout/authored/how-to-create-a-missing-words-question-mr1sgfmw.png)

### 7. Points
Enter the total score for the question here.

![Points](/img/helpscout/editors/how-to-create-multiple-choice-questions-9.png)

Select the **“Shuffle Last Option”** checkbox to randomize the position of the last answer option in the question.

### 8. Correct Answer Feedback
Use **Correct Answer Feedback** if you want to add additional information to a question. This is optional.

![Use Correct Answer Feedback if you want to add additional information to a question. (Optional)](/img/helpscout/editors/how-to-create-a-missing-words-question-8.png)

The content in the **Correct Answer Feedback** and **Reviewer Notes** sections is shown to participants after they submit their answers and can include text, images, or videos.

### 9. Reviewer Notes
You can add any additional details/feedback on the question or answer options for the other trainers/editors.

![You can add any additional feedback on the question or answer options.](/img/helpscout/editors/how-to-create-a-missing-words-question-9.png)

### 10. Categories
Every question must be assigned at least **one category**. Categories are predefined by your administrator. You may select multiple categories if applicable.

![Every question must be assigned at least one category. Categories are predefined by your administrator. You may select multiple categories if applicable.](/img/helpscout/editors/how-to-create-a-missing-words-question-10.png)

## Note on Copying Questions from Microsoft Word
Microsoft Word adds certain hidden characters and markups to every document. If you directly copy text from a Word document, these hidden artifacts can create problems, especially on mobile devices. To avoid this issue, if you are copying any question from a Word document, please follow these steps:

1. Copy the text from the Word Document into Notepad. This removes all hidden markups.
2. Then copy the text from Notepad into SmartWinnr → Question box.

## Question Content
Questions can incorporate a variety of content elements, including plain text, videos, images, hyperlinks, and audio, to enhance clarity, engagement, and learning effectiveness. Following are different Content Options available for questions:

* **Text** - If it's just a text question, then you can simply write it in the box.

* **Videos** - You can pose a question through a video as well. Choose videos of products, product demonstrations, customer feedback, customer stories, and ask questions based on that. Click on the video icon. You can add your own videos to the question or videos from YouTube, Box, and Brainshark.

    [Learn how to add different videos to a question](https://smartwinnr-help-docusaurus-production.up.railway.app/modules/quiz/features/how-to-add-a-video-to-a-question)

* **Images** - Images or pictures can be sent with a question by clicking on the image icon. Then click on the 'Choose File' button to select the required files from the computer.

    [Learn how to add an image to the question](https://smartwinnr-help-docusaurus-production.up.railway.app/modules/quiz/features/how-to-add-an-image-to-the-question)

* **Links** - You can also insert a link with the question by clicking on the links icon.

    [Learn how to embed a link into the question](https://smartwinnr-help-docusaurus-production.up.railway.app/modules/quiz/features/how-to-embed-a-link-into-the-question)

* **Audio** - You can also send an audio file with the question by clicking on the Audio icon.

    [Learn how to embed an audio file into the question](https://smartwinnr-help-docusaurus-production.up.railway.app/modules/quiz/features/how-to-add-an-audio-file-to-a-question)

## Change Language
You can create the question different language. By default the question gets created in English. So, if you would like to create the question in any other language, change the language of the question by clicking on the **"Change Language"** button at the top right corner of this section. It will open the following popup:

![Change the language of the question by clicking on the "Change Language" button.](/img/helpscout/editors/how-to-create-multiple-choice-questions-13.png)

Choose the language to which you want to change and then click on the "Save" button.

Once you have filled in the required details of the question, click on the **'Save New Question'** button to save the question.
