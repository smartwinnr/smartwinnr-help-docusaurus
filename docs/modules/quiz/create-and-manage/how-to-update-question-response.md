SmartWinnr enables editors to update or correct a user’s response in a quiz. When a response is modified, the system automatically recalculates the score and updates the revised results across all related analytics and reports.

To update or correct a user’s response, follow the steps outlined below:

1. Navigate to **Editor > Questions and Quizzes**.
2. Click on **Analytics** for the relevant quiz.

    You will be redirected to the quiz analytics page.

![You will be redirected to the quiz analytics page.](/img/helpscout/editors/how-to-update-question-response-1.png)

Scroll down to the bottom and click on analytics icon towards the user you wish to update the response.

![Scroll down to the bottom and click on analytics icon towards the user you wish to update the response.](/img/helpscout/editors/how-to-update-question-response-2.png)

Click on "**Change Response"** against the user response where you want to make the change

![Click on "Change Response" against the user response where you want to make the change](/img/helpscout/editors/how-to-update-question-response-3.png)

Update the current response  and click on " **Save"**

![Update the current response and click on " Save"](/img/helpscout/editors/how-to-update-question-response-4.png)

Then the response will be updated successfully along with the score as given below.

![Then the response will be updated successfully along with the score as given below.](/img/helpscout/editors/how-to-update-question-response-5.jpg)
