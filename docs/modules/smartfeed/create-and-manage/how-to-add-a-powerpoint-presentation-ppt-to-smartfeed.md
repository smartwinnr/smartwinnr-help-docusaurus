SmartWinnr allows you to include PowerPoint presentations in your SmartFeeds, enabling you to share rich, structured content such as product knowledge, training material, or feature highlights with your users.

### Steps to Add a PowerPoint Presentation

* #### Step 1: Navigate to SmartFeeds

* Log in to the **SmartWinnr Admin Portal**.
* Go to **EDITOR > LEARNING AND KNOWLEDGE > SmartFeeds > View All SmartFeeds**.
* Click **Add New SmartFeed** to create a new SmartFeed.

* #### Add a SmartFeed Title

    Provide a clear title for the SmartFeed. This title will be visible to all recipients and should summarize the presentation content.

![Provide a clear title for the SmartFeed. This title will be visible to all recipients and should summarize the presentation content.](/img/helpscout/editors/how-to-add-a-powerpoint-presentation-ppt-to-smartfeed-1.png)

#### Step 2: Choose the Content Type

* On the **Create SmartFeed** screen, select **Presentation** as the content type.
* This option allows you to upload PowerPoint files (`.ppt`  or `.pptx` ).

If you don't see the 'Presentation' option in your view, please write to [support@smartwinnr.com](mailto:support@smartwinnr.com) for the privilege to add presentations to your SmartFeed.

* Once selected, the Presentation upload screen will appear.

![If you don't see the 'Presentation' option in your view, please write to support@smartwinnr.com for the privilege to add presentations to yo](/img/helpscout/editors/how-to-add-a-powerpoint-presentation-ppt-to-smartfeed-2.png)

##### Upload as a ZIP File (Recommended)

Uploading a ZIP file is recommended as it ensures that the slide order remains exactly the same as in the original PowerPoint.

To upload using ZIP format:

1. Convert each slide of the PowerPoint presentation into image files (`.jpeg` , `.jpg` , or `.png` ).
2. Save all the slide images into a single folder.
3. Compress (zip) the folder.
4. Select the **ZIP** radio button in the upload section.
5. Upload the ZIP file to SmartWinnr.

##### How to Convert a PowerPoint to JPEG Images

1. Open the PowerPoint presentation.
2. Go to **File > Export**.
3. Choose **Export presentation images**.
4. Select **JPEG** as the file format.
5. Save the images to a folder on your system after that you can zip that folder

After uploading the ZIP file, select the appropriate **SmartFeed category** and click **Save New SmartFeed** to proceed.

**Important:**

Once a  ZIP file is uploaded, SmartWinnr automatically converts it back into **Presentation Mode**, allowing recipients to view the SmartFeed as a slide-based presentation.

### Step 3: Assign the SmartFeed to Users

After uploading the ZIP file, click **Save SmartFeed**.

You will be redirected to the **Send SmartFeed** page.

On this page, you can:

* View the list of available participants
* Select users based on your requirement

Once you have selected the required users, click **Send SmartFeed** to successfully assign the SmartFeed to the selected participants.

![Once you have selected the required users, click Send SmartFeed to successfully assign the SmartFeed to the selected participants.](/img/helpscout/editors/how-to-add-a-powerpoint-presentation-ppt-to-smartfeed-3.png)

#### Configure SmartFeed Settings

1. Set the **Send Date** (Send Now or Schedule Later).
2. Set the **Expiry Date** (Never expire or choose a specific date).

[Learn more on send date, expiry date and category value for SmartFeed.](https://smartwinnr.helpscoutdocs.com/article/27-how-to-create-a-smartfeed)
