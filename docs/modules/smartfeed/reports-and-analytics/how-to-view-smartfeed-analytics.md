SmartFeed Analytics provides insights into the **number of views** for each SmartFeed created by you, as well as for SmartFeeds to which you have access. This helps you understand the reach and engagement of your shared content.

### How to Access SmartFeed Analytics

1. From the **left menu bar**, navigate to **SmartFeeds**.
2. Click on **SmartFeed Analytics**.

Once opened, you will see **two tabs** within the SmartFeed Analytics section, allowing you to view and analyze SmartFeed performance.

* **My SmartFeeds Tab** displays analytics for the SmartFeeds created by you.
* **Others Tab** displays analytics for SmartFeeds created by other editors.

![](/img/helpscout/editors/how-to-view-smartfeed-analytics-1.png)

You can also search for a specific SmartFeed using the **Search** box or filter SmartFeeds by selecting a date range from the **Date** filter.

By selecting the **Image** icon, you can download and save the SmartFeed Analytics graph as an image for future reference.

![](/img/helpscout/editors/how-to-view-smartfeed-analytics-2.png)

## Downloading SmartFeed Reports

You can download SmartFeed analytics reports in multiple formats to analyse user engagement in detail.

## Available Report Types

![](/img/helpscout/editors/how-to-view-smartfeed-analytics-3.png)

### **Download Report by User**

This report provides a consolidated view of SmartFeed engagement by user. Each user appears **once** in the report, along with the **total number of views** recorded for the SmartFeed.

In the **Download Report by User**, you can view user details, the associated SmartFeed information, and the **total number of views** recorded for each user.

![](/img/helpscout/editors/how-to-view-smartfeed-analytics-4.png)

### **Download Report by View**

This report provides a **view-level breakdown** of SmartFeed engagement. Each row represents a single view by a user for a SmartFeed, allowing you to track views **based on date**.

In the **Download Report by View**, each view is captured in a **separate row**, along with the corresponding **view date**.

![](/img/helpscout/editors/how-to-view-smartfeed-analytics-5.png)

### **Download Report by SmartFeed Comments**

This report includes **user details along with their comments** on the SmartFeed, enabling you to review feedback and interactions associated with each SmartFeed.

In the **Download Report by SmartFeed Comments**, you can view the SmartFeed’s basic details, the comment details, and the information of the user who posted each comment.

![](/img/helpscout/editors/how-to-view-smartfeed-analytics-6.png)
