## How to view All SmartFeeds that I have created in SmartWinnr?

SmartWinnr allows editors to view all SmartFeeds that they have created, as well as SmartFeeds created by other editors to which they have access. This helps you manage, review, and reuse SmartFeeds efficiently.

### Steps to View All SmartFeeds

1. Log in to the **SmartWinnr Admin Portal**.
2. Navigate to **EDITOR > LEARNING AND KNOWLEDGE > SmartFeeds**.
3. Click on **View All SmartFeeds**.

You will be directed to the **SmartFeeds listing page**, which contains two tabs:

### My SmartFeeds

* Displays all SmartFeeds created by you.
* You can view details such as title, content type, status, sent history, and expiry.
* From this tab, you can view, edit, resend, expire, or delete SmartFeeds based on your access permissions.

![](/img/helpscout/editors/how-to-view-all-the-smartfeeds-1.png)

## How to View SmartFeeds You Have Access To?

SmartWinnr allows you to view SmartFeeds created by other editors, provided you have the necessary access permissions.

### Steps to View Accessible SmartFeeds

1. Navigate to **SmartFeeds > View All SmartFeeds**.
2. Switch to the **Other SmartFeeds** tab.

This tab displays all SmartFeeds created by other editors that you have been granted access to. From here, you can view the SmartFeed details and, if permitted, access related analytics and history.

![This tab displays all SmartFeeds created by other editors that you have been granted access to. From here, you can view the SmartFeed detail](/img/helpscout/editors/how-to-view-all-the-smartfeeds-2.png)
