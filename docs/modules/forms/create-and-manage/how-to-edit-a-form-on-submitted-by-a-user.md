An Editor who has access to the form can edit the submission made by a user

Goto Left Menu >> Editor >> Forms >> View All Forms.

![Goto Left Menu >> Editor >> Forms >> View All Forms.](/img/helpscout/editors/how-to-edit-a-form-on-submitted-by-a-user-1.jpg)

Select the form you want to edit and go to the analytics page for the form by clicking on the Analytics icon.

![Select the form you want to edit and go to the analytics page for the form by clicking on the Analytics icon.](/img/helpscout/editors/how-to-edit-a-form-on-submitted-by-a-user-2.jpg)

Then select the user whose form you want to edit and click on the view icon on its right side.

![Then select the user whose form you want to edit and click on the view icon on its right side.](/img/helpscout/editors/how-to-edit-a-form-on-submitted-by-a-user-3.jpg)

Click on the EDIT icon at the top right corner to edit the form.

![Click on the EDIT icon at the top right corner to edit the form.](/img/helpscout/editors/how-to-edit-a-form-on-submitted-by-a-user-4.jpg)

Make the changes you want to make here on this page.

![Make the changes you want to make here on this page.](/img/helpscout/editors/how-to-edit-a-form-on-submitted-by-a-user-5.jpg)

Click on Submit icon when the editing is done.

![Click on Submit icon when the editing is done.](/img/helpscout/editors/how-to-edit-a-form-on-submitted-by-a-user-6.jpg)

This is how you should edit the form on behalf of the user.

Note: If the form has - Need Approval Enabled, then the form can be edited by the editor only before its approved/rejected by the approver.

![Note: If the form has - Need Approval Enabled, then the form can be edited by the editor only before its approved/rejected by the approver.](/img/helpscout/editors/how-to-edit-a-form-on-submitted-by-a-user-7.jpg)

When you click on the View icon, here on this page you can also Approve/Reject the form as well.
