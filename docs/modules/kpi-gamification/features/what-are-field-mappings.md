Step 2 in the setup process involves mapping fields from your data sources to the KPIs that you have defined in SmartWinnr. SmartWinnr processes the raw data from the data sources based on these mappings. This is a crucial step in the setup process.

Below are the steps to view existing mapping:

Go to ADMIN > KPI Settings > Field Mappings.
