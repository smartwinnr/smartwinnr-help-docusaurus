SmartWinnr allows you to create two types of achievements - Badges and Certificates. Follow the below-given steps to create a Badge.

Go to Left Menu>>Editor>>GAMIFICATION AND ENGAGEMENT>>Badges And Certificates

![Go to Left Menu>>Editor>>GAMIFICATION AND ENGAGEMENT>>Badges And Certificates](/img/helpscout/editors/how-to-create-badges-1.png)

This page shows various Badges and Certificates that are already existing,

The editor can view Badges and Certificates individually by selecting the type and then by clicking on the search button.

## Adding Achievement

Click on the "Add Achievement" in the top right-hand corner of the above image, following page will open.

![Click on the "Add Achievement" in the top right-hand corner of the above image, following page will open.](/img/helpscout/editors/how-to-create-badges-2.png)

This page gives options for adding various types of achievements. The editor can select the type of achievement by clicking on the "Achievement Type". After selecting the achievement type(whether It be a Badge or Certificate).

Choose the Badge from the drop-down menu to create a Badge. The editor needs to input the appropriate achievement title corresponding to the achievement.

![Choose the Badge from the drop-down menu to create a Badge. The editor needs to input the appropriate achievement title corresponding to the](/img/helpscout/editors/how-to-create-badges-3.png)

The editor can also write a brief description of the achievement in the "Description" area. Then upload an appropriate image corresponding to the achievement type.

After all the fields are successfully filled Click on "Create Achievement", the new achievement will be added.

Once the Badge is getting created, you have to assign the user to whom you would like to give the Badge.

Click on the 'Assign' option to assign the badge to the users.

![Click on the 'Assign' option to assign the badge to the users.](/img/helpscout/editors/how-to-create-badges-4.png)

It will redirect to the user list page, where you can select the users for whom you would like to share it. Here, you can filter with a group and you can easily assign it to the list of users belonging to the specific group.

![It will redirect to the user list page, where you can select the users for whom you would like to share it. Here, you can filter with a grou](/img/helpscout/editors/how-to-create-badges-5.png)

Click on the 'Assign' button at the bottom to assign it to the users.

You can also Bulk assign the badge if you have list of many users that you can not assign manually.

Click on the **Hamburger Menu>>Bulk Assign.**

![Click on the Hamburger Menu>>Bulk Assign.](/img/helpscout/editors/how-to-create-badges-6.jpg)

Once the page is opened you need to download the template by selecting the Badge name from the "**Achievements**" drop down and the Email Id in the "**User Identifier**" Drop down.

![Once the page is opened you need to download the template by selecting the Badge name from the "Achievements" drop down and the Email Id in ](/img/helpscout/editors/how-to-create-badges-7.jpg)

Once the template is downloaded just input the email ID's of the users you want to assign the badge, save the CSV file and upload it under the **Choose File** option.

Once it is assigned, you can also see the list of users for whom the badge has been awarded. You can click on the **Hamburger Menu>>Assignment History.**

![Once it is assigned, you can also see the list of users for whom the badge has been awarded. You can click on the Hamburger Menu>>Assignment](/img/helpscout/editors/how-to-create-badges-8.png)

![Once it is assigned, you can also see the list of users for whom the badge has been awarded. You can click on the Hamburger Menu>>Assignment](/img/helpscout/editors/how-to-create-badges-9.png)

Click on the **View Logs** option to see the list of users for whom the badge has been awarded.
