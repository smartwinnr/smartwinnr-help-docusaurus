Only the Org Admin access can add the observers to the competition leaderboards.

Navigate to **EDITOR > GAMIFICATION AND ENGAGEMENT > Competitions.**

1: Choose the competition for which you want to add observers.

![1: Choose the competition for which you want to add observers.](/img/helpscout/editors/how-to-add-observers-to-the-competition-leaderboard-1.jpg)

2: After selecting the competition, select the 3rd window "View Leaderboards"

![2: After selecting the competition, select the 3rd window "View Leaderboards"](/img/helpscout/editors/leaderboards-of-a-competition-7.png)

3: Click on "Manage Leaderboard Observers" and select the users who you want to add as observers. This option helps you to select the observers for all the leaderboards in the contest. It will open the below page where you can select the leaderboards and the observers to add.

![3: Click on "Manage Leaderboard Observers" and select the users who you want to add as observers. This option helps you to select the observ](/img/helpscout/editors/leaderboards-of-a-competition-12.png)

Click on the **Save** icon at the bottom to complete the activity. Then, the below popup will come,

![Click on the Save icon at the bottom to complete the activity. Then, the below popup will come,](/img/helpscout/editors/how-to-add-observers-to-the-competition-leaderboard-2.png)

Click on **Yes** button to add them as observers.

Once saved, the selected observers will be displayed on the **Observers** button corresponding to that leaderboard.

![Once saved, the selected observers will be displayed on the Observers button corresponding to that leaderboard.](/img/helpscout/editors/leaderboards-of-a-competition-13.png)

If you want to assign different observers to the leaderboards, click the same button for the respective leaderboard and follow the same steps to add the observers.
