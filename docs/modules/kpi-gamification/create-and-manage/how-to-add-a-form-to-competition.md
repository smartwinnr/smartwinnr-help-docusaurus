### Steps to Add a Form to a Competition

1. In the **fourth step** of creating a competition, select **Form** from the **“Select Task”** dropdown.
2. Enter a **name** for the form task.
3. Set the **start and end dates** for the task.

Below is a screenshot illustrating how to add a **Form** to a competition:

![](/img/helpscout/editors/how-to-add-a-form-to-competition-1.png)

4. Select the **desired Form** you want to add to the competition.
5. Click the **“Create”** button. The Form will be **successfully added** to the competition.
