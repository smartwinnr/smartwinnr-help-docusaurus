### Steps to Add a Quiz to a Competition

1. In the **fourth step** of creating a competition, select **Quiz** from the **“Select Task”** dropdown.
2. Enter a **name** for the quiz task.
3. Assign the **points** to be awarded for completing the quiz.
4. Set the **start and end dates** for the quiz task.

Below is a screenshot of adding a quiz to a competition:

[![](/img/helpscout/editors/how-to-add-a-quiz-to-a-competition-1.png)](https://s3.amazonaws.com/helpscout.net/docs/assets/5fcc5d22d580ce55a38b4a61/images/600a894d2e764327f87c1452/file-tEBJHTwpgp.png)

5. **Choose Participants Manually:** Check the **“Choose Participants Manually”** checkbox to manually select the participants for the quiz. This allows you to **exclude specific users** from the participant list if necessary.
6. **Assign Points:** Specify the **points for correct answers** to be awarded to quiz takers. Enter the desired value in the provided field.
7. **Create the Quiz:** Click the **“Create”** button. You will be redirected to a page where you can **create a new quiz** and add it to the competition.

For detailed instructions on creating a quiz, refer to the following articles:

[How to create an automatic quiz?](https://smartwinnr.helpscoutdocs.com/article/49-how-to-create-an-automatic-quiz)

[How to create a manual quiz?](https://smartwinnr.helpscoutdocs.com/article/50-how-to-create-a-manual-quiz)

Clicking on the "Duplicate Quiz" button opens the following pop-up from where you can select an existing quiz and duplicate it.
