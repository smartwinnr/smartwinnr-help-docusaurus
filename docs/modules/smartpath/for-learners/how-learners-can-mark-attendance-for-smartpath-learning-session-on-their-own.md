Learners can self-mark their attendance for learning sessions in SmartPath directly from the **User View** using the built-in scanner option.

### Steps to Mark Attendance

1. Login to your account in SmartWinnr.
2. Switch to user view, if you have both User and Manager role in SmartWinnr.
3. Navigate to **SmartPath**.
4. Select the **SmartPath** that contains the learning session for which you need to mark attendance.

![Select the SmartPath that contains the learning session for which you need to mark attendance.](/img/helpscout/managers-and-team-members/how-learners-can-mark-attendance-for-smartpath-learning-session-on-their-own-1.png)

5. On the SmartPath screen, tap the **Scanner icon** available at the **top-right corner**

![On the SmartPath screen, tap the Scanner icon available at the top-right corner](/img/helpscout/managers-and-team-members/how-learners-can-mark-attendance-for-smartpath-learning-session-on-their-own-2.png)

5. Scan the QR code shared for the learning session.
6. Once the scan is successful, your attendance will be marked automatically.

![Once the scan is successful, your attendance will be marked automatically.](/img/helpscout/managers-and-team-members/how-learners-can-mark-attendance-for-smartpath-learning-session-on-their-own-3.png)

### Important Notes

* The scanner option is available only for SmartPaths that include **learning session modules**.
* Ensure you have the correct SmartPath selected before scanning the QR code.
