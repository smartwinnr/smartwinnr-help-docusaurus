SmartWinnr allows you to duplicate an existing survey so that you can reuse the structure, questions, logic, and settings without having to recreate everything from scratch. This feature helps save time when creating similar surveys for different audiences or periods.

> **Note:** Only Trainers or  **EDITORS** with appropriate access permissions can duplicate surveys.

Follow the below steps to duplicate a survey through SmartWinnr.

Go to **EDITOR > LEARNING AND KNOWLEDGE > Surveys**. Select any "Survey" which you want to duplicate.

![Go to EDITOR > LEARNING AND KNOWLEDGE > Surveys. Select any "Survey" which you want to duplicate.](/img/helpscout/editors/how-to-duplicate-a-survey-and-survey-template-1.png)

The following page will be displayed. Now select the hamburger menu which is present on the top right corner of the page.

![The following page will be displayed. Now select the hamburger menu which is present on the top right corner of the page.](/img/helpscout/editors/how-to-duplicate-a-survey-and-survey-template-2.png)

From the following menu select the Duplicate option.

![From the following menu select the Duplicate option.](/img/helpscout/editors/how-to-duplicate-a-survey-and-survey-template-3.png)

Select Yes to create a duplicate copy of the selected survey.

![Select Yes to create a duplicate copy of the selected survey.](/img/helpscout/editors/how-to-duplicate-a-survey-and-survey-template-4.png)

Before saving the duplicated version:

* Rename the survey title to reflect its new purpose.
* Update the **description**, **start and end dates**, **visibility settings**, or any other relevant fields.
* Modify questions if needed to fit the new context.

Once all necessary updates are completed:

* Click **Save** to save the duplicated survey.

![Once all necessary updates are completed:](/img/helpscout/editors/how-to-duplicate-a-survey-and-survey-template-5.png)
