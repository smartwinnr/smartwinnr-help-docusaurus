## What are the different question types in a Survey/Survey Template

There are 8 Question Types available in a survey template.

Please note that all these questions come with a checkbox "Mark as mandatory". By default, this checkbox is checked which makes it mandatory for the survey takers to answer the question. You can uncheck this if you do not want to make the question mandatory.

### 1\. Single Select

Choose this question type when you want the survey taker to choose only one answer to the question.

### ![Choose this question type when you want the survey taker to choose only one answer to the question.](/img/helpscout/editors/what-are-the-different-question-types-in-a-survey-survey-template-1.png)**2\. Multiple Select**

Use the question type ‘Multiple Select’ when you need to add a question where the survey takers can select multiple answers.

###

![Use the question type ‘Multiple Select’ when you need to add a question where the survey takers can select multiple answers.](/img/helpscout/editors/what-are-the-different-question-types-in-a-survey-survey-template-2.png)

### “Others” Option in Dropdown Questions

SmartWinnr provides an **“Others”** option in **Dropdown** question types.

This option allows participants to select **“Others”** when none of the predefined dropdown values are applicable to them.

![This option allows participants to select “Others” when none of the predefined dropdown values are applicable to them.](/img/helpscout/editors/what-are-the-different-question-types-in-a-survey-survey-template-3.png)

3. ### Linear Scale

This type of question is useful when you want to ask your participants to rate something. It provides a question with a linear scale. You can change the range of this linear scale upto 10. Also you can provide label to the lowest and hightest rating possible.

###

![This type of question is useful when you want to ask your participants to rate something. It provides a question with a linear scale. You ca](/img/helpscout/editors/what-are-the-different-question-types-in-a-survey-survey-template-4.png)

4. ### Short Answer

Use the question type ‘**Short Answer**’ when you need to add a question where the survey takers can provide descriptive answer.

###

![Use the question type ‘Short Answer’ when you need to add a question where the survey takers can provide descriptive answer.](/img/helpscout/editors/what-are-the-different-question-types-in-a-survey-survey-template-5.png)

5. Matrix

Use the question type '**Matrix**' when you want to take customer's feedback about the service/ service facilitators.

###

![Use the question type 'Matrix' when you want to take customer's feedback about the service/ service facilitators.](/img/helpscout/editors/what-are-the-different-question-types-in-a-survey-survey-template-6.png)

In a **Matrix question**, SmartWinnr provides two display options to define how the answer choices appear. One of these options is **Rating** and **Checkbox.**

When the **Rating** option is selected, you can configure a rating scale based on your requirement, such as **1 to 5** or **1 to 10**. This allows respondents to rate each statement in the matrix using that rating scale.

The rating scale question will appear as shown below.

![The rating scale question will appear as shown below.](/img/helpscout/editors/what-are-the-different-question-types-in-a-survey-survey-template-7.png)

When the **Checkbox** option is selected, users can choose one or multiple options based on their requirement.

![When the Checkbox option is selected, users can choose one or multiple options based on their requirement.](/img/helpscout/editors/what-are-the-different-question-types-in-a-survey-survey-template-8.png)

The **Checkbox** question will be displayed as shown below.

![The Checkbox question will be displayed as shown below.](/img/helpscout/editors/what-are-the-different-question-types-in-a-survey-survey-template-9.png)

To create a rating question with **subjective ratings**, uncheck both **“Is Rating”** and **“Is Checkbox.”**

Once disabled, a text field will appear where you can enter the subjective rating options separated by commas (for example: _Poor, Average, Good, Very Good_) and click on save option near to the field.

After entering the options and save click the **Submit** button to save the question.

![After entering the options and save click the Submit button to save the question.](/img/helpscout/editors/what-are-the-different-question-types-in-a-survey-survey-template-10.png)

The Rating question with Subjective Rating will be displayed as shown below.

![The Rating question with Subjective Rating will be displayed as shown below.](/img/helpscout/editors/what-are-the-different-question-types-in-a-survey-survey-template-11.png)

6.Use the question type '**Date**' when you want to ask date-related questions to the survey taker.

### ![6.Use the question type 'Date' when you want to ask date-related questions to the survey taker.](/img/helpscout/editors/what-are-the-different-question-types-in-a-survey-survey-template-12.png)7\. Dropdown

Choose the question type as '**Dropdown**' when you have to give a dropdown question to your survey takers.

![Choose the question type as 'Dropdown' when you have to give a dropdown question to your survey takers.](/img/helpscout/editors/what-are-the-different-question-types-in-a-survey-survey-template-13.png)

### 8.**File**

The **File Question Type** allows respondents to upload a file as their answer to a question. This is useful when you need participants to submit supporting documents or evidence instead of selecting or typing a response

* Uploading **documents** (PDF, Word, Excel)
* Submitting **images** (JPEG, PNG)
* Sharing **videos or audio files**
* Providing **proof or supporting material** for an activity or assessment
