Survey Analytics allows you to monitor and analyze responses collected from your surveys. This feature provides insights into user participation and feedback patterns, enabling you to evaluate results effectively.

Follow the below steps to view analytics in Survey.

Go to **EDITOR > LEARNING AND KNOWLEDGE > Surveys.**

![Go to EDITOR > LEARNING AND KNOWLEDGE > Surveys.](/img/helpscout/editors/survey-analytics-1.png)

Under the ' **My Surveys'** section select Survey  **Type=Survey** which is present very next to the Date Range.

![Under the ' My Surveys' section select Survey Type=Survey which is present very next to the Date Range.](/img/helpscout/editors/survey-analytics-2.png)

Now under Analytics, click on the analytics icon against the survey for which you want to view the analytics.

You will be taken to the following page, where the analytics of the selected survey is displayed.

![You will be taken to the following page, where the analytics of the selected survey is displayed.](/img/helpscout/editors/survey-analytics-3.png)

Filter By Response: There is separate analytics for Dropdown and Multi-select questions in Survey. Here you can filter with a particular Dropdown/Multiselect question and select one of the options of the question for which you want to see the analytics.

![Filter By Response: There is separate analytics for Dropdown and Multi-select questions in Survey. Here you can filter with a particular Dro](/img/helpscout/editors/survey-analytics-4.png)

The following are the different types of graphs displayed for different question types in the survey.

## 1\. Single Select Question

The graph related to this question type is displayed using a pie chart by indicating the responses in different colors and the percentage of people who responded to that particular option.

![The graph related to this question type is displayed using a pie chart by indicating the responses in different colors and the percentage of](/img/helpscout/editors/survey-analytics-5.png)

## 2\. Multi-Select Question

The graph related to this question type is displayed using a bar graph, the question options are displayed in the y-axis and the percentage of users who responded in shown in the x-axis from the scale of 0 to 1 (where 0=0% and 1.0=100%).

![The graph related to this question type is displayed using a bar graph, the question options are displayed in the y-axis and the percentage ](/img/helpscout/editors/survey-analytics-6.png)

## 3\. Linear Scale Question

The graph related to this question type is displayed using a bar graph, from low to high scale value is shown in the x-axis, and the percentage of users responded is displayed on the y-axis(where 0=0% and 1.0=100%).

![The graph related to this question type is displayed using a bar graph, from low to high scale value is shown in the x-axis, and the percent](/img/helpscout/editors/survey-analytics-7.png)

## 4\. Matrix Question

Matrix question graphs are useful to represent the ratings and these are shown in the x-axis, whereas the y-axis consists of the options given for a particular category.

![Matrix question graphs are useful to represent the ratings and these are shown in the x-axis, whereas the y-axis consists of the options giv](/img/helpscout/editors/survey-analytics-8.png)

## 5\. Date Question

The graph related to this question type is also displayed using a bar chart. The date is shown on the x-axis and the percentage scale is shown on the y-axis. The percentage of response is indicated with color as shown below.

![The graph related to this question type is also displayed using a bar chart. The date is shown on the x-axis and the percentage scale is sho](/img/helpscout/editors/survey-analytics-9.png)

## 6\. Dropdown Question

The graph related to this question type is displayed using a donut chart by indicating the responses in different colors and the percentage of people who responded to that particular option.

![The graph related to this question type is displayed using a donut chart by indicating the responses in different colors and the percentage ](/img/helpscout/editors/survey-analytics-10.png)

#### **Aggregate Report**

This type of report consists of the survey question, responses, and percentage of response for each answer.Click on the Download Report Button which is present towards the right corner. The report will be downloaded as an excel sheet.

![This type of report consists of the survey question, responses, and percentage of response for each answer.Click on the Download Report Butt](/img/helpscout/editors/survey-analytics-12.png)

The **Aggregate Report** displays a consolidated view of responses for each question, along with the corresponding **percentage distribution**.
