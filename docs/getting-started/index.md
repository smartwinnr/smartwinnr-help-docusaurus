## Quick Setup Guide

### 1. First Login

Download the SmartWinnr app or navigate to [app.smartwinnr.com](https://app.smartwinnr.com) and log in using the credentials provided by your administrator. Complete your profile setup to personalize your SmartWinnr experience.

**Need help getting started?** Check out our [Account Setup & Login](./account-setup-login) guide for detailed instructions.

### 2. Dashboard Overview

Your dashboard provides a comprehensive overview of:

- **Active Competitions** - Current contests and your ranking
- **Pending Quizzes** - Assignments waiting for completion  
- **Learning Progress** - SmartPath and microlearning advancement
- **Performance Metrics** - Key statistics and achievements

### 3. Navigation Structure

The platform is organized into these main sections:

| Section | Purpose | Key Features |
|---------|---------|-------------|
| **Dashboard** | Performance overview | Real-time metrics, notifications |
| **Quiz** | Knowledge assessments | Interactive questions, immediate feedback |
| **Competitions** | Gamified challenges | Leaderboards, prizes, team contests |
| **Microlearning** | Bite-sized content | SmartFeeds, progressive learning |
| **Reports** | Analytics & insights | Detailed performance analysis |
