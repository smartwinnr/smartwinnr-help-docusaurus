With the Admin role in SmartWinnr, you can change the password for users from the division and subdivisions you have access to.

To change the password goto **Left Menu > Admin > Manage Users.** This will show you a list of users to whom you have access. Search for the user with the various search option like name, email, meta tags available to you.

![](/img/admin/helpscout-image-ac3278bc.png)

Click on the ' **Change Password**' button against the user for whom you would like to change the password. The password you select needs to match with your organizational password policy as defined in the SmartWinnr system.

The user will not receive any mail on changing the password through this method. You need to communicate the changed password to the user separately.
