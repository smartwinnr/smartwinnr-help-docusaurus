SmartWinnr allows you to set up Learning sessions as part of the SmartPath. There are two types of learning sessions available in SmartWinnr:

* Classroom Learning Session - This is basically an offline classroom session where the training is scheduled on SmartWinnr and users have to attend the classroom training session physically.
* Virtual Learning Session - This is an online session conducted on any virtual platform like Zoom, Google meet, etc where the learning session can be scheduled under the Smartwinnr platform.

To conduct and schedule the learning session, the editor should create a **SmartPath Module** under which the offline classroom session or onlineﾂVirtual Sessionﾂcan be scheduled. Creating a SmartPath Module for learning sessions is similar to the way we create quizzes/surveys/SmartFeeds in SmartPath Module.

Please follow the below steps to create a module for the Learning session.

Go toﾂ **EDITOR** >ﾂ**LEARNING AND KNOWLEDGE** >ﾂ**SmartPaths** > **View All Modules**

On this page, click on theﾂ' **Create Module**'ﾂbutton at theﾂ _top right corner_ﾂof the screen.

![](/img/learning/helpscout-image-87b452d7.jpg)

Here add a title to your Module which can be the name for the Training session or the topic for discussion. You can also write details about the training in the Description and add a cover image based on the topic of training/company.ﾂ Click on **Save Module** and proceed to the next step.

![](/img/learning/helpscout-image-2d051084.jpg)

The first half of the module is now created and to proceed further,ﾂ _Click_ﾂon theﾂ'**Add Segment**ﾂ_button_ﾂat theﾂ_top right corner_ﾂof the page.

![](/img/learning/helpscout-image-75db493e.jpg)

To create a segment that is basically the classroom/virtual session,ﾂ _choose_ﾂthe option-ﾂ**Learning Session**ﾂin the '**Select Segment Type**'ﾂdropdown as shown below.

![](/img/learning/helpscout-image-91d0f8df.png)

After selecting the Learning Session,ﾂ _enter_ﾂtheﾂ'Name'ﾂfor the segment and you can select how many points the user will get after completing this segment under **Point Distribution>> On Completion** and click on theﾂ'**Create**'ﾂbutton. This will successfully create your segment forﾂLearning Session. Refer to the below picture for reference.

![](/img/learning/helpscout-image-758c8031.jpg)

Once the learning segment is created, it will take you to the following page where you can fill in the details of the Learning Session.

Enter aﾂ **Title** for your session and you can also mention the details about the training in the description (Optional).

Now **select the session type** where you can find two options: **Virtual Session and Classroom Session.**

* **Virtual Session:** Virtual session says that you are going to conduct your training virtually through some online platform. When you choose the session type as Virtual Session, in the further steps, the system will ask you to provide the link where the meetings can be conducted virtually. (Example: Microsoft Teams, Google Meet)
* **Classroom Session:** This is like the normal classroom sessions where the trainer and trainees come in tow a classroom or conference Hall to attend the training.ﾂWhen you choose the session type as ClassroomﾂSession, in the further steps, the system will ask you to mention the address of the place where the session will be conducted.

![](/img/learning/helpscout-image-ba1733dc.jpg)

You also have the option to send a calendar invitation for the scheduled training session so that the user can be prepared for the training. The editor needs to enable the **Send Calendar Invitation** to send the invitation.

![](/img/learning/helpscout-image-3ccd359d.jpg)

When the Editor enables the calendar invite, the user will receive the invitation as shown below.

![](/img/learning/helpscout-image-814aefcd.png)

Then click on the **Save button** to complete creating the learning session.

![](/img/learning/helpscout-image-7edabaac.jpg)

Once your Module is created with theﾂLearning sessionﾂsegment, you need to attach this module to a SmartPath to assign this to the users who will take the training.

[Click here](https://help.smartwinnr.com/article/248-how-to-create-a-learning-session-for-smartpaths) to learn more on how to attach a module with a learning session to SmartPath.
