# Quiz & Assessments

Create engaging assessments to test knowledge, measure learning progress, and drive performance across your organization.

## Key Features

### Assessment Types

- **Knowledge Quizzes**: Test understanding of products, processes, or policies
- **Video Quizzes**: Interactive assessments embedded in video content
- **Survey Quizzes**: Gather feedback and opinions
- **Timed Assessments**: Add urgency with time limits

### Question Formats

- Multiple Choice (Single/Multiple answers)
- True/False questions
- Fill in the blanks
- Long answer questions
- Interactive questions (Drag & drop, Hotspot, Matching)
- Numeric and calculation questions

## Module Overview

### Creating Assessments

Learn how to design and build effective assessments that engage learners and measure knowledge accurately.

### Question Management

Master the art of question creation with comprehensive guides for all question types, multimedia integration, and question bank management.

### Assessment Administration

Manage the lifecycle of your assessments from assignment and distribution to access control and security settings.

### Assessment Analytics

Analyze performance data to identify learning gaps, measure effectiveness, and optimize your assessment strategy.

## Getting Started

1. **Plan Your Assessment**: Define learning objectives and choose appropriate question types
2. **Create Questions**: Use our question management tools to build engaging content
3. **Configure Settings**: Set up timing, scoring, and security options
4. **Assign & Launch**: Distribute to learners and monitor progress
5. **Analyze Results**: Use analytics to measure effectiveness and improve

## Best Practices

- **Clear Objectives**: Align questions with specific learning outcomes
- **Varied Question Types**: Mix different formats to maintain engagement
- **Immediate Feedback**: Provide explanations to reinforce learning
- **Progressive Difficulty**: Start easy and gradually increase complexity
- **Regular Updates**: Refresh content based on performance data
