## How to view All SmartFeeds that I have created in SmartWinnr?

You can also view all the SmartFeeds sent by going to **Smartfeeds > View All SmartFeeds.**

![](/img/learning/helpscout-image-f1998c5a.png)

## How to view All SmartFeeds that I have given access to in SmartWinnr?

To view all the SmartFeeds that you have access to, goto **Smartfeeds > View All SmartFeeds** >> Change the tab to '**Other SmartFeeds**'.

![](/img/learning/helpscout-image-ae1d143d.png)
