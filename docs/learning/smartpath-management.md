# SmartPath Management

## What are SmartPaths?

A SmartPath is a structured learning path that guides individuals through a sequence of educational content. Each SmartPath contains various modules that learners must complete in order to master a particular topic or skill set.

SmartPaths are extremely useful when you need to take learners through concepts in a specific sequence to teach them certain skills or lessons effectively.

![SmartPath Structure](/img/learning/helpscout-smartpath-structure-0d281cde.png)

### How SmartPaths Work

SmartPath creation follows a **bottom-up approach**:

1. **Create Modules** that will be part of the SmartPath
2. **Add Segments** to each module (SmartFeeds, Quizzes, Surveys, Coaching)
3. **Encapsulate Modules** into a complete SmartPath
4. **Deploy** the SmartPath to learners

**Example Scenario**: Improving sales effectiveness
- **SmartPath**: "Sales Effectiveness Excellence"  
- **Module 1**: "Effective Opening" (contains 1 SmartFeed, 1 Quiz, 1 Survey)
- **Module 2**: "Objection Handling" (contains 2 SmartFeeds, 2 Quizzes, 2 Coachings)

## Creating SmartPaths

### Step 1: Create Modules for SmartPath

Before creating a SmartPath, you must first create the modules that will be included:

1. Navigate to **EDITOR > LEARNING AND KNOWLEDGE > SmartPaths**
2. Click **"Create Module"**
3. Define module details:
   - Module name and description
   - Learning objectives
   - Target audience
   - Estimated completion time
4. Save the module structure

### Step 2: Add Learning Sessions to SmartPath Modules

Learning sessions are the building blocks of modules:

1. Open your created module
2. Click **"Add Learning Session"**
3. Choose session type:
   - **SmartFeed sessions**: Content delivery
   - **Quiz sessions**: Knowledge assessment
   - **Survey sessions**: Feedback collection
   - **Coaching sessions**: Skill development
4. Configure session parameters
5. Set completion criteria

### Step 3: Create the Complete SmartPath

Once modules are ready:

1. Click **"Create SmartPath"**
2. Add SmartPath information:
   - Title and description
   - Learning objectives
   - Target completion time
   - Prerequisites (if any)
3. Add modules to the SmartPath in the desired sequence
4. Set progression rules (linear, adaptive, or free-form)
5. Configure completion requirements

### Step 4: Create SmartPath with Learning Sessions

For comprehensive SmartPaths that include interactive sessions:

1. Enable **"Learning Sessions"** in your SmartPath
2. Schedule live or self-paced sessions
3. Add session materials and resources
4. Set up attendance tracking
5. Configure session reminders and notifications

## Managing SmartPaths

### Editing SmartPaths

To modify existing SmartPaths:

1. Go to **SmartPaths > Manage SmartPaths**
2. Select the SmartPath to edit
3. Modify components:
   - Update module content
   - Adjust sequence or prerequisites
   - Change completion criteria
   - Update target audiences
4. Save changes and republish

### Archiving SmartPaths

To archive SmartPaths without deleting them:

1. Navigate to **SmartPath Management**
2. Select the SmartPath to archive
3. Click **"Archive SmartPath"**
4. Choose archival options:
   - Remove from active learner lists
   - Preserve learner progress data
   - Set future unhide date if needed
5. Confirm archival

### Unhiding Archived SmartPaths

To restore archived SmartPaths:

1. Go to **Archived SmartPaths** section
2. Find the SmartPath to restore
3. Click **"Unhide"** or **"Restore"**
4. Configure restoration settings:
   - Update content if needed
   - Reassign to learner groups
   - Set new completion deadlines
5. Republish to learners

## Learning Session Management

### Setting Up Interactive Sessions

For SmartPaths that include live learning components:

1. **Schedule Sessions**: Set dates, times, and duration
2. **Add Presenters**: Assign trainers or subject matter experts  
3. **Prepare Materials**: Upload presentations, handouts, resources
4. **Configure Technology**: Set up virtual meeting tools if remote
5. **Send Invitations**: Notify participants with session details

### Managing Session Attendance

Track and manage learner participation:

1. **Mark Attendance**: Record who attended each session
2. **Track Participation**: Monitor engagement during sessions
3. **Handle Absences**: Set policies for missed sessions
4. **Send Reminders**: Automatic notifications for upcoming sessions
5. **Follow-up Actions**: Provide makeup sessions or alternative content

### Sending Absent Reminders

To improve attendance and engagement:

1. Access **Session Management**
2. Identify participants who missed sessions
3. Click **"Send Absent Reminder"**
4. Customize reminder message:
   - Highlight importance of session content
   - Provide makeup options
   - Include session recordings if available
5. Schedule follow-up reminders if needed

## SmartPath Analytics

### Viewing SmartPath Analytics

Monitor learning effectiveness and progress:

1. Go to **SmartPaths > Analytics**
2. View comprehensive metrics:
   - **Completion Rates**: Percentage of learners finishing paths
   - **Progress Tracking**: Individual and group advancement
   - **Time to Complete**: Average duration for path completion
   - **Module Performance**: Which modules are most/least effective
   - **Engagement Levels**: User interaction with content

### Key Performance Indicators

Track essential SmartPath success metrics:

- **Enrollment Numbers**: How many learners are assigned
- **Start Rate**: Percentage who begin the SmartPath
- **Completion Rate**: Percentage who finish successfully  
- **Average Score**: Performance across assessments
- **Time Investment**: Hours spent learning
- **Satisfaction Ratings**: Learner feedback scores

### Using Analytics for Improvement

- **Identify Bottlenecks**: Where learners struggle or drop off
- **Optimize Content**: Improve underperforming modules
- **Adjust Pacing**: Modify session timing based on completion data
- **Personalize Paths**: Create adaptive learning experiences
- **Measure ROI**: Demonstrate learning program effectiveness

## Taking SmartPaths (Learner Perspective)

### How Learners Access SmartPaths

Participants can access assigned SmartPaths through:

1. **SmartWinnr Mobile App**: Download and access on-the-go
2. **Web Interface**: Complete sessions on desktop/laptop
3. **Offline Mode**: Download content for offline learning
4. **Cross-device Sync**: Continue progress across devices

### Learner Experience

When taking a SmartPath, users will:
1. See their assigned learning path with clear progression
2. Complete modules in the required sequence
3. Access diverse content types (videos, documents, quizzes)
4. Track their own progress and achievements
5. Receive notifications for new content or deadlines

## Best Practices

### SmartPath Design

- **Clear Learning Objectives**: Define what learners will achieve
- **Logical Sequence**: Order content for optimal comprehension
- **Appropriate Pacing**: Balance challenge with achievability
- **Diverse Content**: Mix videos, text, quizzes, and interactions
- **Regular Assessments**: Check understanding throughout the path

### Content Management

- **Version Control**: Track updates and changes
- **Regular Reviews**: Keep content current and relevant
- **Quality Assurance**: Test all links, media, and interactions
- **Mobile Optimization**: Ensure excellent mobile experience
- **Accessibility**: Make content available to all learners

### Learner Support

- **Clear Instructions**: Explain how to navigate and complete paths
- **Technical Support**: Provide help for platform issues
- **Content Support**: Offer assistance with learning materials
- **Progress Monitoring**: Check in with struggling learners
- **Feedback Mechanisms**: Allow learners to report issues or suggestions

## Reusability and Efficiency

SmartPaths and their components are designed for maximum reusability:

- **Modules**: Use across multiple SmartPaths
- **Learning Sessions**: Repurpose for different audiences  
- **Assessments**: Reuse quizzes and evaluations
- **Content**: Share SmartFeeds and resources
- **Templates**: Create standardized path structures

This modular approach saves time and ensures consistency across your learning programs while allowing for customization when needed.
