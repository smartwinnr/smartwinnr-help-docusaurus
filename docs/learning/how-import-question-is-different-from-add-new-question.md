The following are the main differences between creating questions using the 'Add New Question' method and  **bulk upload** method.

1. 'Add New Question'  allows you to create only one question at a time. Whereas Bulk Upload lets you create multiple questions at a time and upload them.
2. You can have any number of options for a multi-select question if it is created using the 'Add New Question' method.  On the other hand, using the Bulk Upload you can have at most 6 options for the multi-select question.
3. You can create Hotspot question using the 'Add New Question' method. It is not possible to create a Hotspot question using Bulk Upload.
