If you are an instructor for the learning session in a SmartPath created in SmartWinnr, you should mark the attendance of the trainees who attended the learning session. Only after marking the attendace for trainees, the system will change the status of SmartPath learning session to 'Completed' for each trainee.

To mark the attendance for the learning session you conducted as part of SmartPath, kindly follow the below steps:

1\. **Log** in to your account **from SmartWinnr App/ Web view**

2\. **Switch to user view in SmartWinnr** in case if you have both user role and manager role.

![](/img/learning/helpscout-image-2f4f3f24.png)

3\. Goto **Left Menu >> SmartPath.**

4\. Here, you will find **a tab called learning session** if you are an instructor for any learning session created in SmartWinnr.

5\. Select the learning session for which you would like to mark the attendance now.

![](/img/learning/helpscout-image-62273d74.png)

6\. This will give you the list of users added to the selected learning session. **Select the users who have attended the learning session** as follows:

![](/img/learning/helpscout-image-9264daaa.png)

Click on the '**Mark Attendance**' button to successfully mark the attendance of select users as 'Present'.
