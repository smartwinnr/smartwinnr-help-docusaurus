Only the user with the EDITOR role can add the SmartFeed into Smartwinnr.

Login to SmartWinnr Backend Portal.

Go to EDITOR > LEARNING AND KNOWLEDGE > SmartFeeds > View All SmartFeeds.

It will open the following page:

![](/img/learning/helpscout-image-6d66e1cb.jpg)

After selecting the division, choose the content type as Recording. After entering the title, select which type of category do you want to record.

**Screen:** It allows you to record your screen. ( For example, you can record the steps to follow the process like Data entry)

**Video:** It allows you to record your video. (For example, you can record yourself explaining the process steps)

**Audio:** It allows you to record your voice in audio format.

Change the settings as per your requirements and select the required categories and save the SmartFeed.
