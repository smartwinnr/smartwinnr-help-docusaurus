## What are the different question types in a Survey/Survey Template

There are 7 Question Types available in a survey template.

Please note that all these questions come with a checkbox "Mark as mandatory". By default, this checkbox is checked which makes it mandatory for the survey takers to answer the question. You can uncheck this if you do not want to make the question mandatory. 

### 1\. Single Select:

Choose this question type when you want the survey taker to choose only one answer to the question.

### ![](https://file2.api.drift.com/drift-prod-file-uploads/e2b0%2Fe2b04b4f651b9a3b93f097d8dcc8ce46/qstn.png?mimeType=image%2Fpng)**2\. Multiple Select**

Use the question type ‘Multiple Select’ when you need to add a question where the survey takers can select multiple answers.

### ![](https://file2.api.drift.com/drift-prod-file-uploads/f28e%2Ff28ee20b05b215f7cb036ed20e235bad/multi.png?mimeType=image%2Fpng)3\. Linear Scale

This type of question is useful when you want to ask your participants to rate something. It provides a question with a linear scale. You can change the range of this linear scale.

### ![](https://file2.api.drift.com/drift-prod-file-uploads/1179%2F11799a004720a75cbaebc29f724e0d26/linear.png?mimeType=image%2Fpng)4\. Short Answer

Use the question type ‘**Short Answer**’ when you need to add a question where the survey takers can write a text for the answer.

### ![](https://file2.api.drift.com/drift-prod-file-uploads/f14b%2Ff14bd00f8d8133e71fde34ca7eae65a2/short%20ans.png?mimeType=image%2Fpng)5\. Matrix question

Use the question type '**Matrix**' when you want to take customer's feedback about the service/ service facilitators.

### ![](https://file2.api.drift.com/drift-prod-file-uploads/61c4%2F61c4e4ddd526b0cadf5d04736ec07a1e/matrix.png?mimeType=image%2Fpng)6\. Date

Use the question type '**Date**' when you want to ask date-related questions to the survey taker.

### ![](https://file2.api.drift.com/drift-prod-file-uploads/6eef%2F6eef749277d6d15df66892b3ef3ee2c2/date.png?mimeType=image%2Fpng)7\. Dropdown

Choose the question type as '**Dropdown**' when you have to give a dropdown question to your survey takers.

![](https://file2.api.drift.com/drift-prod-file-uploads/e67e%2Fe67e178ae8f642cf0468f51e21ba7fe9/dropdown.png?mimeType=image%2Fpng)
