SmartWinnr allows the managers to edit the form submitted by their reporters.

The first step is to log in through the manager's view and click on the forms from the left side menu.

Refer to this link https://help.smartwinnr.com/article/96-quiz-report-for-managers

Select the form for which you want to edit the form submission.

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5fcc5d22d580ce55a38b4a61/images/60bb7bc5cfd1573abf70c9d4/file-qmdpkvtnEc.png)

Choose the duration and select the form with pending status. ( You cannot edit the form submission once it's approved )

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5fcc5d22d580ce55a38b4a61/images/60bb7c1b90777e23fb8ada53/file-CzSGom8tAV.png)

Click on the EDIT icon at the right side corner

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5fcc5d22d580ce55a38b4a61/images/60bb7c7d90777e23fb8ada54/file-Uizklvl31g.png)

Here you can edit the date, visit type and the necessary changes can be made here.

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5fcc5d22d580ce55a38b4a61/images/60bb7ce6f6392f0aaff521dc/file-m1RFiuLSt7.png)

After editing, please click on the save button to complete the editing.
