This feature lets you set a reminder and schedule it to be delivered to the users about the status of the Task. To schedule a reminder, click on the " **Add**" button at the top right corner of the Schedule Reminder section. It will open the following popup.

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5fcc5d22d580ce55a38b4a61/images/5ffc684566df373cab708131/file-64mTFGQicL.png)

**Send Type:** This field lets you decide if you want to send the reminder on a specific date or if you want to send it depending on the end date of the quiz.  

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5fcc5d22d580ce55a38b4a61/images/6001ce8b2e764327f87bf27b/file-sQNxgwze8b.png)  
**Date to send:** Here, select the date and time when you want to send the reminder.

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5fcc5d22d580ce55a38b4a61/images/6001cea1b9a8501b295d0f17/file-pck0ITeFLy.png)  
**Send before end (In Days):** If you select "Depends on Quiz End Date" then you will see this field. Here, enter the number of days before the end date of the quiz when you want to send the reminder.

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5fcc5d22d580ce55a38b4a61/images/6001ceb4cfe30d219ccd8425/file-40emDppAsh.png)  
**Send To:** Here you can choose to send it to "Everyone" or a "Group".

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5fcc5d22d580ce55a38b4a61/images/6001cec7cfe30d219ccd8426/file-WrW7yp8Kqu.png)  
**Filter By:** Here you can filter the quiz takers to whom you want to send the reminder. There are 3 options to choose from-- "All", "Completed", "Not Completed".

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5fcc5d22d580ce55a38b4a61/images/6001cee31c64ad47e4b6fbc1/file-QboIHdADz5.png)  
**Teams:** If you have selected "Group" in the Send To field then you have to select the teams to whom you want to send the reminder.

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5fcc5d22d580ce55a38b4a61/images/6001cf07b9a8501b295d0f19/file-blj8WZJcHx.png)  
  
**Message:** This will display the message that will be sent in the reminder. This field will have an auto-generated message but you can change it according to your requirement.

  
![](https://s3.amazonaws.com/helpscout.net/docs/assets/5fcc5d22d580ce55a38b4a61/images/6001cf38c64fe14d0e1fb634/file-fi5xOpugst.png)

  
Once you are done filling all these fields for the reminder, click on the "  **Save**" button. This will add the reminder successfully to the quiz.  
  
Once you fill all the details on this page, click on the  "Save and Add Questions" to save the quiz.
