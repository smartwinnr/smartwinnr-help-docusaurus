##### A Missing Words Question allows the participants to choose one answer to a particular question from a list of options provided.

##### Below are the steps to create a missing words question.

1.  Go to EDITOR > LEARNING AND KNOWLEDGE > Questions and Quizzes > Question Bank.

2.  Click on the 'Add New Question' button at the _top right corner_.
    
    ![](https://s3.amazonaws.com/helpscout.net/docs/assets/5fcc5d22d580ce55a38b4a61/images/642f5a941274e915f2f92fa6/file-zdXD3F59yS.png)
    

3.  On the 'Create new Question' select the 'Question Type' as Missing Words.
    
    ![](https://s3.amazonaws.com/helpscout.net/docs/assets/5fcc5d22d580ce55a38b4a61/images/642f5bd91274e915f2f92fa7/file-ada24rvLg3.png)
    

4.  Enter your question in the ‘Question’ field.
    
    ![](https://s3.amazonaws.com/helpscout.net/docs/assets/5fcc5d22d580ce55a38b4a61/images/642f5be44cd1ab01bbe8ba22/file-k7vFLKrp2I.png)
    

_Here you can define the number of missing words._

_Enter the question text and in place of the missing word just place the Placeholder by using the button shown below:_

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5fcc5d22d580ce55a38b4a61/images/642f5bef7bf4bb61c0117099/file-aZpHz38Nza.png)

5.  As we have given only 1 missing word, we will define 1 group for that and provide a list of answers and choose the correct answer.
    
      
    
    ![](https://s3.amazonaws.com/helpscout.net/docs/assets/5fcc5d22d580ce55a38b4a61/images/642f5bfa4cd1ab01bbe8ba23/file-ufRdS2aRB0.png)
    
      
    
    Question Feedback
    
    Use ‘Correct Feedback’ and **'Incorrect Feedback'** if you want to add additional information to a question.  (Optional) 
    
    _The feedback is visible only after the participant has completed answering the questions in a quiz. You can add text, videos, or images to feedback._
    
    ![](https://s3.amazonaws.com/helpscout.net/docs/assets/5fcc5d22d580ce55a38b4a61/images/642f5c0ba3236b1edbce9379/file-JVPWFwC1do.png)
    
    In SmartWinnr, _every question requires at least one category_. Categories are pre-defined by your administrator. You can select one or more categories that best defines a question.
