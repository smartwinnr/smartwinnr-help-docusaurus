Once the SmartFeed has been sent, there is an option to edit it if anything needs to be changed.

When you click on a particular SmartFeed, You will find an ' **Edit'** option at the _top right_ corner.

[![](https://s3.amazonaws.com/helpscout.net/docs/assets/5fcc5d22d580ce55a38b4a61/images/5ff0a7de495a886ca782954b/file-sOpQVX53IJ.png)](https://s3.amazonaws.com/helpscout.net/docs/assets/5fcc5d22d580ce55a38b4a61/images/5ff0a7de495a886ca782954b/file-sOpQVX53IJ.png)

Clicking on it will open the below edit screen where the SmartFeed can be edited.

[![](https://s3.amazonaws.com/helpscout.net/docs/assets/5fcc5d22d580ce55a38b4a61/images/5ff0a89f495a886ca782954e/file-q1QVuq9S3y.png)](https://s3.amazonaws.com/helpscout.net/docs/assets/5fcc5d22d580ce55a38b4a61/images/5ff0a89f495a886ca782954e/file-q1QVuq9S3y.png)

Once you have made the necessary changes to the SmartFeed, click on the ' **Update**' button at the bottom. This updates the SmartFeed successfully.
