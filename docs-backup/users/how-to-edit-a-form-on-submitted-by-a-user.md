An Editor who has access to the form can edit the submission made by a user

Goto Left Menu >> Editor >> Forms >> View All Forms.

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5fcc5d22d580ce55a38b4a61/images/60bb9b1fcfd1573abf70c9ee/file-xuamVC68Jn.jpg)

Select the form you want to edit and go to the analytics page for the form by clicking on the Analytics icon.

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5fcc5d22d580ce55a38b4a61/images/60bb449b90777e23fb8ada23/file-7NtdJwJoZ9.jpg)

Then select the user whose form you want to edit and click on the view icon on its right side.

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5fcc5d22d580ce55a38b4a61/images/60bb412d90777e23fb8ada21/file-MyfhnVGhGA.jpg)

Click on the EDIT icon at the top right corner to edit the form.

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5fcc5d22d580ce55a38b4a61/images/60bb41bb90777e23fb8ada22/file-VHVIpNxOoR.jpg)

Make the changes you want to make here on this page.

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5fcc5d22d580ce55a38b4a61/images/60bb423cd656bf05f300bc5d/file-OqcaChn1TP.jpg)

Click on Submit icon when the editing is done.

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5fcc5d22d580ce55a38b4a61/images/60bb43bef6392f0aaff5219a/file-hqgFFqmkWY.jpg)

This is how you should edit the form on behalf of the user.

Note: If the form has - Need Approval Enabled, then the form can be edited by the editor only before its approved/rejected by the approver.

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5fcc5d22d580ce55a38b4a61/images/60bb6d8cfca38f033a7da60d/file-3fqemyYQJk.jpg)

When you click on the View icon, here on this page you can also Approve/Reject the form as well.
