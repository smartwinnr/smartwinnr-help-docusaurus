Only the Org Admin access can add the observers to the competition leaderboards.

Go to EDITOR > GAMIFICATION AND ENGAGEMENT > Competitions.

**Step 1: Choose the competition for which you want to add observers.**

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5fcc5d22d580ce55a38b4a61/images/60b605a6afcffb24193600b6/file-zlp7xWY9r6.jpg)

**Step 2: After selecting the competition, select the 3rd window "View Leaderboards"** 

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5fcc5d22d580ce55a38b4a61/images/60b605fdafcffb24193600b9/file-vEd1vlqCDX.jpg)

**Step 3: Click on the "Manage Observers" and select the users who you want to add as observers.**

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5fcc5d22d580ce55a38b4a61/images/60b606729c887a0dfc555358/file-bac6ULktEO.jpg)Click on the save icon at the bottom to complete the activity.
