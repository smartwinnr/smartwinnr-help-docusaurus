Only the user with the EDITOR role can add new form submission.

The first step is to click on forms in the left side menu. EDITOR>>PERFORMANCE>>Forms>>View all forms.

Click on Create Form icon on the top right side and create a new form.

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5fcc5d22d580ce55a38b4a61/images/60b7806f4dda6972e093188d/file-aoo9mclI73.jpg)

Fill up the form title and description and set up the duration of the form. You can click on Advanced and check-in those boxes which are needed.

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5fcc5d22d580ce55a38b4a61/images/60b781af4dda6972e0931898/file-lZVFO8gdlA.jpg)

**Is Edit Allowed**\- By enabling this will allow the manager to make changes after approving any form submission. 

**Needs Approval**\- Enabling this will allow the manager to approve the submission before getting evaluated.

You can also select a header by uploading the PDF Header Image.

You can also select or change the submission type as None, Frequency( How often the forms are submitted) and the Count (Number of forms that are submitted).

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5fcc5d22d580ce55a38b4a61/images/60b7844f4dda6972e09318ad/file-pnLoRocijd.jpg)

You can also change the Notification Type to None, Email, and app notifications. Also, you should enter the title for Form Submission Feedback.

![](https://s3.amazonaws.com/helpscout.net/docs/assets/5fcc5d22d580ce55a38b4a61/images/60b785972246b50b7f3916e1/file-GVC24UBERO.jpg)

In the 2nd step, you should **Add Field** by entering the **Name**, **Field Type**, and **Data Type**.
