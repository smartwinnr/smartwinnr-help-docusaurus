🔧 Setup Required

To use the CMS, you need to configure GitLab OAuth:

  1. Go to GitLab → User Settings → Applications
  2. Create a new application with:
    • Name: SmartWinnr Help CMS
    • Redirect URI: https://help.smartwinnr.com/admin/
    • Scopes: api
  3. Copy the Application ID and update static/admin/config.yml
  4. Replace your_gitlab_app_id with your actual Application ID
  5. Refresh this page
